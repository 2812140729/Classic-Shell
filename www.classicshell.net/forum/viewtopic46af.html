Post subject:  bootup and windows key
Love the Classic Shell. It fixes a lot of what I wanted after upgrading to Windows 8. I have two requests regarding preferences I have:
1. I would like the Windows Key to continue to operate as in Windows 8. When I'm in the Win 7 desktop and press the Windows Key I would like it to take me to the Win 8 Start screen rather than bringing up the Win 7 start menu.

2. When I bootup, I would like to end up at the Win 8 Start screen rather than the Win 7 desktop.

How can I accomplish both or either of these modifications?
Author:  Gaurav [ Thu Jan 10, 2013 8:52 am ]
Post subject:  Re: bootup and windows key
1. Right click the Start button and click Settings. From Basic Settings tab or Controls tab (if you are viewing All Settings), change "Windows Key opens" to "Windows Start Screen".

2. From same Basic Settings tab, locate a setting called "Skip Metro screen" and uncheck it. If you are viewing All Settings, the setting is located on the Windows 8 Settings tab.

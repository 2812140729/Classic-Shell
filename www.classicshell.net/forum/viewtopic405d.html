Post subject:  Re: View Control Panel by Category
Note that in Classic Shell 4.x Windows 7 style, the option to open Control Panel in category view in Windows Explorer is on the Main menu tab of Start Menu settings but you can also use the search box to find it.

For classic styles of the menu in Classic Shell 4.0 and in Classic Shell 3.6.x, the same setting is on the Special Items tab of Start Menu settings.

Note that this only affects how the Control Panel opens in Explorer when it is launched from the Start Menu, and not how the Control Panel shows inside the Start menu.

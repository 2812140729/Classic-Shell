Author:  Ivo [ Sat Aug 20, 2016 8:42 pm ]
Post subject:  Re: Install errors ClassicExplorerXX
This error means "access is denied". It is hard to know access to what, but I'm guessing the registry. Possibly the registry permissions are messed up.
Open an admin command prompt in the Classic Shell directory and type the command "regsvr32 ClassicExplorer32.dll". See if you get any more detailed error message.
Author:  Gaurav [ Sat Aug 20, 2016 10:58 pm ]
Post subject:  Re: Install errors ClassicExplorerXX
If you have an anti-virus, it could be preventing Classic Shell files from being registered. Try to temporarily disable it and install again.

Or as Ivo said, it can be a registry permissions issue.

Hi shpkong.

Really wild designs on the net, that can be a custom menu image.

Here is what I was talking about before.
Picture is example about the white text on white menu background, with possible tint option on/off.
It might not be the best way to get effect. Looks a bit muddy.
You would need to make extra search pattern mask 47, with red at 200/255.
I put this text in Skin 1 folder:


OPTION TINTED="Tinted custom Image",0,CUSTOM_IMAGE_LEFT,0


[TINTED AND CUSTOM_IMAGE_LEFT AND COLOR_BLACK OR COLOR_GREEN OR COLOR_BLUE OR COLOR_RED OR COLOR_CUSTOM]
Main_pattern_mask=#C80000
Main_pattern_search_mask=#C80000
Main_pattern_jump_mask=#C80000
Search_pattern_mask=47


For Skin 2 folder with the round corners, another main pattern mask 48, with 200/255 red,
would need to be made as well:

[TINTED AND CUSTOM_IMAGE_LEFT AND COLOR_BLACK OR COLOR_GREEN OR COLOR_BLUE OR COLOR_RED OR COLOR_CUSTOM]
Main_pattern_mask=48
Main_pattern_search_mask=48
Main_pattern_jump_mask=48
Search_pattern_mask=47

If you want more background images, you can simply make many beautiful background images by yourselves thanks to a great software abstractcurves.

Whose offical website is: http://abstractcurves.com/

Here is a simply guidence on how to make a background images with a few clicks.

First, launch abstractcurves and click new image.


Then, a windows will pop out for you to choose size, transparecy and so on. The recommend background image size is 700*1400, but if you are using a high dpi monitor you should set the size a little lager so that the images will not show repeatedly.


Here goes the main windows. Here is a very simple introduction of the basic features. If transparent background opion is enabled, the back ground image will behave like Twister alpha only. Use a custom dark color.png made by juniper7.

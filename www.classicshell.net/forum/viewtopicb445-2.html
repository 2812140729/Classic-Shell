Post subject:  Re: Optical error
No, that's not how menus work. Only custom accelerators are underlined, default accelerators are not.
If you want a letter to be underlined you need to use & in the name. For example rename Computer to &Computer. Or Sta&rtseite.

Post subject:  Re: Optical error
The name for the start screen item comes from the StartMenuL10N.ini file because it is localized in all languages. You can change it there, but it may break during the next upgrade.

Another way to change the name is to open the properties and change the name from startscreen.lnk into whatever you want. The next time you open the start menu you will get both until you disable the "Start Screen shortcut" from the settings. The downside here is that you will get the default icon and it won't match the tiles in your start screen.

So basically there is no ideal solution to rename it. I still don't get why you would want that though, just to get the underline? The S key would work even if it is not underlined.

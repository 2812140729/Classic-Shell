Author:  Gaurav [ Mon Jun 17, 2013 7:34 am ]
Post subject:  Re: Pograms and Features
1. If you are using the new Windows 7 style, go to Customize Start Menu tab.
2. Press the Insert key to add a new item. Position it where you want
3. It will be added as Administrative Tools. Change it to "Custom".
4. Double click the icon on the left of 'Custom'.
5. In the Link field, type: shell:ChangeRemoveProgramsFolder
6. Click OK.

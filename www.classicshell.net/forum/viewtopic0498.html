I understand what it is like.
If it's the first time to use the settings, this might help:
After you have copied or downloaded the image to your computer,
right click on the start button to get a small menu, pick settings.

Select taskbar tab. Select "Customize taskbar"
halfway down the list select "Taskbar texture"
Click on the three dashes.
Go to where ever you have the downloaded taskbar image and open it in that window.
That should load it. Just need to set it up.
If you are using taskbar on bottom select "Horizontal stretching" to "Stretch"
"Vertical stretching" to "Borders" with border sizes 2,0
That makes the two top pixels not stretch, and the bottom pixel stretch.
If it does not look correct after that, then wrong image was used for bottom.


Each image that people have uploaded requires some adjusting to get it looking correct.
Some work right away using default settings.
Hope this helps.

I love Classic Shell. This is very good software that transforms ugly Windows 10 to a beautiful desktop. Now if only functionality of "Old New File Explorer" could be added in Classic Shell and somehow we could modify Title bars of windows this will be perfect. Even now Classic Shell is amazing.

How to Make Transparent texture from taskbar image
----------------------------------------------
Delete one pixel hard border on top by cropping if needed. Take 1 pixel width only. For patterned taskbars like Windows 7 take whole width of 1920 pixel.
Make new layer above backgound
Make one more layer above this one. Copy backgroud layer (by select all (Ctrl+A), copy(Ctrl+C)) and paste(Ctrl+V) on this layer.
Background Layer visisbility -off
Image -Adjustments -Auto contrast
Image -Adjustments -Desaturate
make top most layer with color Opacity -30%
Export as png

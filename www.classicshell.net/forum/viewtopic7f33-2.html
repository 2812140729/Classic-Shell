There is no way to call the Ctrl+Alt+Del screen from Classic Start Menu, so I don't understand how it is related to Classic Shell.

Also the Windows 10 Start Menu is not modified by Classic Shell.

If Shutdown and Restart have stopped working in the Windows 10 menu as well (and it says "No power options available"), there is something wrong with the power management features of your PC or the drivers. Open Device Manager and make sure all drivers are correctly installed.

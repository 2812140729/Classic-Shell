Post subject:  FYI new W10 update 1803 didn't bork Classic Start menu
My W10 laptop updated today to v. 1803. This seemed like a major update--my laptop threatened to reboot several times--only did so twice. It re-set my mouse and touchpad settings, and Classic Shell announced that it needed to reset for the new OS (or something to that effect); when I clicked OK, Class Start menu re-appeared and is working fine.

The 10 seconds or so of forced useage of the default W10 start menu made me extra relieved to see Classic Start menu back.

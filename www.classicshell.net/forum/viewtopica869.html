The things we could adjust from Advanced Appearance fall into two categories: one that applied to any theme - including those with visual styles and ones that applied only to the "Classic" Windows theme which wasn't based on visual styles. With Classic theme itself removed in Windows 8, some parameters have no effect. Someone in the community needs to make a list of all the things that are still applicable and can still be adjusted. Like icon spacing etc or window borders which are all definitely still changeable. Then maybe someone can write an app. Of course, it won't go into Classic Shell as it has a policy of original features based on its own code only, no registry tweaks. Registry tweaks can break across Windows releases and do break far too often in modern versions of Windows.

The black background is part of the button. That is exactly how the Windows 8.1 button is and the color of the flag is the accent color you set on the Start screen.

For a vertical taskbar, you could reduce the taskbar width by unlocking it so it fits the width of the button. 7+ Taskbar Tweaker (http://rammichael.com/) has a setting 'no_width_limit' in the Advanced Options so your vertical taskbar can be thinner.

I am using it with a tablet which has a touchscreen too. It works for me. Be sure to read these tips to make Classic Shell usable on high DPI display & touchscreens: viewtopic.php?f=5&t=5665

The Start Menu when touch features are enabled, high DPI is enabled and icon sizes increased, works really well with touch. Some menu elements like the split buttons used for Shutdown or jumplists are too small but since it is so customizable, you can increase the size of menu elements or toolbar items by editing the skin or using the Settings.

I modified the Windows 7-style skins shipping in Classic Shell 4.2.5 for you to make more of the Start menu elements easily targetable with a finger (the split buttons). Download the ZIP if you want these bigger buttons and extract it to C:\Program Files\Classic Shell\Skins

Note: These are only Windows 7-style skins. If you use Classic with two columns style, those skins are different, let me know. Also, these skins are copyright of Ivo Beltchev who is the developer of these skins.

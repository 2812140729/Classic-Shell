Author:  Ivo [ Fri Dec 19, 2014 7:08 pm ]
Post subject:  Re: Startmenu on RDS 2012
The start menu has 3 styles - 1 column classic, 2 column classic, and Windows 7. You can switch between them from the "Start Menu Style" tab in the settings.

Some settings apply only to particular styles. When editing the settings using the UI, it hides the settings that don't apply for the current style or OS version.
The policy editor doesn't have such capability to hide or disable particular policies. So the requirements for each setting (which style it applies to, or which OS version) are listed in the description.

Author:  Gaurav [ Mon Jan 14, 2013 3:47 am ]
Post subject:  Re: Show "Hibernate" as default shut down option in box?
First, make sure hibernation is enabled by running this command from an elevated cmd prompt: powercfg /hibernate on

If you wish to make Hibernate the default option in the Shutdown box (the classic Shutdown dialog), open Group Policy (Gpedit.msc -> User Configuration -> Start Menu and Taskbar -> Change Start Menu power button. Set the action there.

You can also make Hibernate the default option in the Start menu itself. For that, open Start Menu settings -> All settings -> Customize Start Menu tab. Locate and double click "ShutdownBoxItem" and from the "Command:" field dropdown, choose 'hibernate'.

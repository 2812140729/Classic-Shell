Right click the Start button -> Settings. Check "Show all settings" and go to the Controls tab. Set "Left click opens" to "Classic Start Menu" and click OK.

If you don't want the Windows logo button and instead want the Classic Shell logo button, open the same settings window and go to the Styles tab and pick a Start button there.

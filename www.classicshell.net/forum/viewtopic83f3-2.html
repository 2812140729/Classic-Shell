I assume you are asking this for Windows 10? The problem is even though the taskbar background can be skinned, the window buttons can't be made to look like Windows 7/8.1. So far, I only know of StartIsBack++ which can theme the taskbar to make it look like Windows 7/8.1. Unfortunately, StartIsBack++ and Classic Shell are not compatible with each other - so you can't use SIB++ for the taskbar and Classic Shell for the menu. Given a choice, I obviously prefer the far more functionally superior Classic Shell Start Menu.

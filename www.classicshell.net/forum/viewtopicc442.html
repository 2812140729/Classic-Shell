Author:  Gaurav [ Sun Nov 15, 2015 9:27 pm ]
Post subject:  Re: [REQUEST] Midnight skin for Classic with two columns.
The Windows 7 style also allows for a multi-column All Programs menu. Select the option "Cascading sub-menu" under "All Programs style" on the Main Menu tab.

An advantage of the Windows 7 style is that it lists search results from top to bottom so they don't jump/change position as more results appear. Also, it has an option to open All programs automatically if you are using the compact treeview.

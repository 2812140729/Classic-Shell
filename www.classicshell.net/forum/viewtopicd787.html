Post subject:  Re: How to make Control Panel open in category view?  
If you are using the classic style then look for a setting called “Use Categories view” and is in the Special Items tab. If you are using the Windows 7 style then it is called “Use Categories view for the Control Panel” and is at the bottom of the Main Menu tab.

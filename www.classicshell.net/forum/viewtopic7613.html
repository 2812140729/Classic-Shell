Classic Shell comes with its own skinning system. Custom visual styles are not supported at all. You can create a Classic Shell Start Menu skin using Resource Hacker similar to the one in the visual style and it will forever work and look the same across all Windows releases. Here is the skin reference: http://www.classicshell.net/tutorials/skintutorial.html

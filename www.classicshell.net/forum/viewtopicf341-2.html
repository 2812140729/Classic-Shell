Author:  Gaurav [ Mon Oct 21, 2013 6:19 am ]
Post subject:  Re: Windows 8 Start Button with Glow (White)
To change the size of any button, go to the "Start Button" tab in Menu settings (You will have to show all settings). The "Custom Button" has a 'Button size' option. 0 means use the width of the image. Change it to match your taskbar's height so it looks like a perfect fit. You will have to click OK after changing the size as the size setting is not live.

Any size you want. Try 16. If it feels too small, then increase the size. This particular image at smaller sizes makes the Windows 8 logo almost stick to the left edge (if taskbar is at the bottom). In that case, you can simply edit it in a free editor like Paint.net and add some transparent pixels to the left of the whole image.

CSM is a great menu for the masses, and the tinkerers.

Would Ivo ever have time to make a little sibling menu,
with only x86 application launcher, for other people?
Like the one anyone can make in the taskbar toolbar,
but with a few changes, for the better?

It would be able to blend the two Win 8 start menu folders using
the virtual folder shell:::{7be9d83c-a729-4d97-b5a7-1b7313c39e0a}
and keep the expanding menu in a box, like Windows Vista/7.
That way, everything is available within a few inches of curser.
Throw in a dedicated launch button anywhere on taskbar.
That's about it. A real simple, fast and clean one.

I tried playing with 3.9.0 beta.
Almost got it with Win7 style, but when I turned off lots of the permanent links,
then the menu flew away again across the screen, with all my installed programs.

There isn't much in the way of simple launchers out there on net.
The closest I could find, but not close enough are:

7stacks.

Can change it's icon and pin to taskbar.
Can't merge user and all menu folders,
unless done with copies in a folder, so won't update.
Some icons a no show.

Handy Start Menu

Can contain menu without flying away.
Icon no show problems.
Launch button only one place on taskbar.

I 'll keep digging.
Still using CSM minimal menu linked with WinX on a daily basis, until something better comes along. Works fine.

Did find a couple more that get to bare bone launchers, but CSM still da best.

I found a sneaky way of keeping the Win 7 style menu contained without flyout,
when all the parked links are not shown.
Duplicated one of the current items into a custom.
GauravK got me thinking from a post of his about the ampersand.
I put "&" as the label. The menu box is now closed with its frame on right side.
It's a little wider but still ok.
Still showed an outline of a parked link box when hovered over, and location error when clicked on.
I put desktop toggle as a link
%windir%\explorer.exe shell:::{3080F90D-D7AD-11D9-BD98-0000947B0257}
so if I do click by mistake, nothing happens.

So now I have two launch menu versions of linking to WinX.
The old classic and the new classic.

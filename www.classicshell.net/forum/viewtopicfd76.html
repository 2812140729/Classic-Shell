Post subject:  Re: Not classic enough, irritating features in Win 7 explore
That is called the Command Bar and Microsoft does not allow it to be disabled unless system files are patched. There is a dangerous way to disable it: http://www.askvg.com/how-to-make-folder-band-auto-hidden-in-windows-vista (not supported by Classic Shell).

Classic Explorer's toolbar has buttons for all Explorer views, you can only add the ones you want.

To apply a view for all folders of a type (General Items, Documents, Music, Pictures), go to Tools menu -> Folder Options -> View tab -> Apply to Folders button.

Libraries can't be disabled but if you use Win+E, it will open at Computer folder.

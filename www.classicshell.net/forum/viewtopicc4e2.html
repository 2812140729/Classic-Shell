Author:  Gaurav [ Mon Oct 24, 2016 6:33 am ]
Post subject:  Re: Search Pane
I know which pane you are referring to however I asked how you were invoking it so I would suggest you to a way to suppress that. Does it appear when you Alt+Tab to the Desktop from a fullscreen game?

Are you using some replacement Alt+Tab switching app or the default Windows one? In Windows 8 and Windows 10, there are many hidden windows - the search pane that appears on the right is one of them. Usually they are hidden in the stock Alt+Tab UI but they show up in third party Alt+Tab apps. It's poor coding on Microsoft's part, they should be excluded from all Alt+Tab user interfaces. I use a replacement called VistaSwitcher which has a feature to exclude hidden windows like this search pane from the Alt+Tab UI so you never see it - it never gets triggered.

I don't think there would be a way to kill or suppress only that window without affecting the rest of the Metro UI.

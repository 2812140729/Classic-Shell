Post subject:  Re: Small aesthetic question
How does this look?
If you need to make tweaks: download Reshacker 3.6 (The newer version is buggy don't bother with it)
Open the .skin7 file with it
Select the first file in the first folder
then hit ctrl+F and type "Main2_padding=8,846,8,9"
The number we are looking for here is the 846, this specifies how far down the second column buttons start
If you need more space, you want a bigger number.
If it is pushing the left column up (aka blank space at the bottom left) then you want a smaller number.
After picking a number compile, and then save (to desktop) then replace the file in your skins folder and restart classic shell.

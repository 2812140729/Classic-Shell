Author:  juniper7 [ Sun Jan 07, 2018 8:49 am ]
Post subject:  Re: Black screen with loading circle after running CCleaner
Oh man, I feel for you.
Deep cleaning registry, or system files can be a gamble.
I never used Start Fresh, as it seems like a half baked way, deleting all the programs one has installed, but keeping other files.

The CC registry cleaner section caused problems for me way back.
I was only using CC as normal recycle file cleaner (not the registry cleaner) after that.

Then a few months ago, I found out certain versions, somehow got infected with super bad malware.
It was in the news.
From then on, I uninstalled it, and have been using Microsoft Disk Cleanup.
Not as fast, but safer security wise.


If you used the registry cleaner section of CCleaner, and if you saved the simple .REG file created
in "My Documents" folder, you might be able to get to it in safe mode, and double click it.

Even though you hate waiting, and would rather throw lemon grenades,
is there any way you can access Windows System Restore in safe mode?
Has been a lifesaver for me.
If you made a Windows restore point sometime before,
it is the full registry backup, and works 75% of time.


If you did what I did when first time using CC software,
I clicked on every file clean box, and screwed up Windows, I had to reinstall operating system.
Hope you don't have to go that far.
There might be other ways to file check for crucial system files, but usually need operating disc for that.

Waiting to get into safe mode, is less work than reinstalling new operating system with usb memory stick!

Good Luck.

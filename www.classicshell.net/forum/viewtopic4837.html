I believe people have successfully used Classic Shell on Server 2012. It is pretty much identical with the desktop version of Windows 8, with some options turned on or off.

From what I hear, it has an alternative system to natively bypass the Metro screen and go to the Desktop, so you might want to disable the Classic Shell feature to skip Metro. However it is probably not going to cause problems even if left enabled.

Post subject:  Re: "Don't display..." not working
Since you have something called "All Programs", then you are using the Windows 7 style.

To remove "Apps" uncheck "Show Metro apps" from the Main Menu tab in the settings.

Everything else in the All Programs list is a shortcut that exists in your Start Menu folders. You can either delete the ones you don't want or hide them. If you delete them, there is a chance some installer or some Windows update will put them back.

Some time ago I made a mod of the Windows XP Luna skin with the right side font/text in bold. I also made some other changes to the skin:

- Left side font and submenu is Segoe UI Semibold. 'Semibold' makes it very readable.
- Increased the arrow padding for split buttons to make it more touch-friendly
- Made the Main_arrow slightly smaller so it looks more elegant for jumplists
- Reduced the font to -9
- The font color in the right column and the Main_arrow color matches the skin variation color

Download it if you are interested.

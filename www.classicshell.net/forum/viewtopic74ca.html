Author:  Jcee [ Mon Nov 27, 2017 4:16 pm ]
Post subject:  Re: Classic Shell only for Transparency (Windows 10)
Yes; Firstly during install only check the 'start menu' component
then right click the start button, go to settings then check 'show all settings' near the top now click the controls tab.. and set everything to 'windows start menu'
Then go to the taskbar tab, and check Customize taskbar, and change the opacity to 0. and hit apply.

This is the process that tries to hook to the main IE frame and show the title bar. Every new tab starts a new process, however they should be short-lived processes. Something is blocking them.
Try logging off and then run IE again. See if these show up again. If they do, we can try few other steps to troubleshoot.

Also, answer these questions: What version of Windows are you using? What is your UAC level? Have you enabled protected mode in IE?

Looks like it is getting stuck in the check for new versions. Maybe you installed a firewall or changes some security settings, so now the internet connection is blocked.

If you are unable to find a way to unblock the check, you can just disable the whole auto-update module. Open the Classic Shell Update tool and uncheck "Automatically check for new versions".

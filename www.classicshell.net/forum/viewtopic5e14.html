Post subject:  Re: PROBLEMS INSTALLING CLASSIC IE
Classic IE restores the page title and status bar features to Internet Explorer 9/10/11.

The Classic Explorer Bar (toolbar) is only added to Windows Explorer, not to Internet Explorer. IE already has a built-in toolbar called the "Command Bar".

The addons need to be enabled for them to work according to the instructions here: http://www.classicshell.net/faq/#explorer_toolbar The reason the Classic Explorer bar shows up in IE's list of addons is because both File Explorer and Internet Explorer share the same architecture for add-ons.

Post subject:  accelerator key display behavior
When classic start menu starts, the menu appears with no accelerator keys displayed. When you the press any key, all the accelerator keys appear underlined as you would expect. Is this intentional? Is there a setting to always display the accelerator key, even when menu is first displayed?
Author:  Ivo [ Tue Jan 08, 2013 10:23 am ]
Post subject:  Re: accelerator key display behavior
That's how menus work. If you activate a menu with the mouse, there are no accelerators. If you start using the keyboard, the accelerators appear. Try it in Explorer. If you activate the File menu with the mouse you see Open. If you activate it with Alt+F, you see Open.

Right. I see. So when I press the start menu key on the keyboard I can see those accelerator keys. Thanks.
Author:  Gaurav [ Wed Jan 09, 2013 9:27 am ]
Post subject:  Re: accelerator key display behavior
Btw, you can go to Ease of Access Center -> Make the keyboard easier to use -> and check the option called "Underline keyboard shortcuts and access keys" to always show accelerator keys even when using the mouse.

If you don't want to always turn them on, there's yet another way. Open the menu using the mouse and press Alt. Now accelerator keys will be underlined.

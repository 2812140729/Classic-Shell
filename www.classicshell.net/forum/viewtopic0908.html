Post subject:  Windows Exploror title bar text and button color
There's an issue with windows 8.1, wherein if a user tries to use a black theme, the text and minimize/maximize buttons on active title bars in the explorer (and inactive tabs and toolbar buttons in firefox) become nearly invisible (they remain black, and there's no way to manually change the colors without using a high contrast theme, which looks ugly and breaks some software). The issue with firefox can be resolved installing a custom theme (I'm using blackfox v2), but there doesn't appear to be any solution for windows explorer and the programs that use windows title bars (after some testing I've found that I don't have any programs that don't use native title bars).

The issue appears to have been resolved in windows 10, so clearly Microsoft is aware of it. They seem to have chosen not to address the issue in windows 8.1. I doubt they intend to in the future either.

Is there any way that the Classic shell Windows Explorer customization application could be extended to allow the customization of title bar text and button color?

(for the record, I tried editing the theme file and the aero.msstyles file with no luck, but there could be something that I'm missing. I also read that the colors could be changed in the registry, but that they'll reset after a reboot).

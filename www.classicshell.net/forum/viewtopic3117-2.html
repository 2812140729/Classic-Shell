BUMP!

I wanted exactly what the original poster of this thread was asking for... the Windows XP Explorer Toolbar icons.

I found either dead links, or icons that looked less than perfect (no disrespect to anyone who tried!).

So, I made my own. I got them as 1:1 as I could, and I think they look pretty dead on!

Here's the link (it's a download from Google Drive): https://goo.gl/MqSStW

The file is also attached to this post!

I bumped the thread because it's a top Google search result and I just wanted for people to be able to find a good result in the future.


From the readme:
These icons are 25x25; the files that are appended _small are 16x16

To use the regular 25x25 icons in Classic Shell Explorer, use these settings
on the "Toolbar Settings" tab:

Big Buttons: Checked
Small icon size: 16
Large icon size: 25
Menu icon size: 16
Text is on the side: Unchecked
Same size buttons: Checked
Resizeable toolbar: Checked

Individually specify the regular 25x25 icon files for each toolbar button.

If you want to use the _small icons, then UNCHECK the "Big buttons" check box,
and individually specify the icons files appended with _small.

(If you use 25x25 and uncheck "Big buttons," or use 16x16 and check "Big
buttons," then the icons will be scaled, reducing image quality.)

I recommend the following sites for .ico conversion from png with
transparencies:
http://image.online-convert.com/convert-to-ico

You can use the following site to batch convert 25x25 icons to 16x16 with good
quality:
http://convertico.com/

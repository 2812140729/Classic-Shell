To bring up the Edit Menu Item dialog, you must double click in the left column of the Toolbar Buttons tab of the Classic Explorer Settings window. Double clicking in the right column will add the items to the left column. Once you double click on any item in the left column, the Label field is one where you must type a name if you want the toolbar command names to show. Otherwise, if you only want the names to show in the tooltip when you hover over them, type the names in the Tip field. The "Command" field will tell you in the Edit Menu Item dialog, which command is which. For the ones that show a generic icon, that is normal - you must supply the icon yourself. You can find some icons here: https://www.dropbox.com/s/ozkb9av08hnol5n/Icons.zip

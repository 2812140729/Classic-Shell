The new version adds support for Windows 8.1 preview.

This is the first version to support Windows 8.1.
This is the last version to support Windows Vista.

Windows Vista is no longer supported because of the addition of:

1. Frequently used app list to Classic Start Menu (whose UserAssist algorithm changed in Windows 7/8 vs Vista)
2. Windows Search 4 integration APIs which are only available in Windows 7
3. Some Windows Installer 5.0 features
4. Very few users on Windows Vista

Next major release is Classic Shell 4.0

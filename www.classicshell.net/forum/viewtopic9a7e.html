Author:  Gaurav [ Thu Sep 28, 2017 6:51 am ]
Post subject:  Re: Support for AutoStart/StartUp-Folder
The actual mechanism for starting a program by placing its shortcut in the Startup folder is a feature of Windows. If you are using Classic Shell's Group Policy Settings to redirect the Start Menu folder, try using the Windows policy settings instead. Then shortcuts placed under the redirected Startup folder might auto-start.

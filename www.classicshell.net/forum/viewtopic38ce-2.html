Author:  Gaurav [ Thu Jul 25, 2013 3:52 am ]
Post subject:  Re: What is the process name of ClassicStartMenu.exe 4.0 Bet  
The start menu process is C:\Program Files\Classic Shell\ClassicStartMenu.exe. Run it with the -exit parameter to exit it. You will need to supply the full path to the EXE in double quotes followed by the -exit parameter to exit it. Also, before relaunching it, a sleep timeout of 2-3 seconds is recommended.

The beta is not that buggy actually. It was very thoroughly tested while it was a private beta.

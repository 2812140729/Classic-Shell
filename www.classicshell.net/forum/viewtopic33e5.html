Post subject:  Single click on shutdown button
I just modified the shutdown button to put the computer to sleep (instead of shutting it down). It takes two clicks to do this. Is there a way to do this with a single click? I'm used to single-clicking that button.
Author:  Gaurav [ Mon Jun 02, 2014 8:17 pm ]
Post subject:  Re: Single click on shutdown button
Right click the Start button -> Settings. Go to the Customize Start Menu tab. Double click the Sleep command (formerly Shut down dialog) which you modified. In the Edit Menu item dialog that comes up, check the option "Split button item" and click OK twice to save the settings. Now you can single click the left part of the button to Sleep/Shutdown whatever and click the right part to show the Shutdown submenu.

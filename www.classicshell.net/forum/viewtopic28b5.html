Since you upgraded from Classic Shell 3.6.8 to the beta, you will have to pick the "Classic with two columns" style now, not the Windows 7 style. What was earlier Windows 7 style is now consolidated under "Classic with two columns" style, while the Windows 7 style is a new one that introduces a menu highly similar to the Windows 7 Start Menu (with All Programs inside an inline treeview).

Once you select Classic with two columns style, you should see your customized Start Menu item order of 3.6.8 which you had set from the 'Customize Start Menu' tab.

These are system items, most of them are not supposed to have context menus.

Post subject:  Bundled Icons
It would be great if Classic Shell could bundle a bunch of additional icons, such as those in the tutorial sections for creating different commands.

maybe just have an additional tab in the Classic Start Menu settings, or just default to a Classic Shell internal install folder containing icons when editing a a menu item..

this way people wouldn't have to rebuild the entire file structure for Icons when they move to a different machine, or copy the settings to a different machine.

Additionally, an internal folder with "BAT" commands would be great, that it defaults to when clicking the "Command:" button on the menu item edit window. Then a list of useful bat files with detailed names could pop up, or better yet, add a separate "Internal Command" drop-down list that lets you select the Bat file function you want to assign to a button, or simply add more buttons ready to go that expand the usefulness a bit.

Icons like "Mic on/off, Chat on/off, Network Enable/Disable, Timer Icon, Music Playlist Icon, 'Search Drive' Icons for Searching in different Drives C: to Z:, Recent Programs Icon, Restore Point Create Icon, CMD as Admin Icon, msconfig icon, appwiz icon, devmgmt.msc icon, diskmgmt.msc icon, Lock desktop icon, refresh network settings.bat icon, flush dns.bat icon, refresh taskbar.bat icon, etc etc

It would just save people alot of time who want to add commands but don't want to have to search for Icons or create their own icons from scratch.

This is not a general purpose Windows support forum for your computer problems. If your issue isn't related to Classic Shell's Start Menu or Explorer addons, you should post it on some big Windows support community such as Neowin, EightForums/SevenForums or MyDigitalLife forum. If you post something unrelated to Classic Shell, post in the Chit Chat forum but note that you may or may not get an answer.

Please read the FAQ here: http://www.classicshell.net/faq/ to troubleshoot your issues. It may save you the trouble of asking questions in the forum which are already answered in the FAQ.

Classic Shell also ships with a help file which may answer your query. You can access it by right clicking the Start button -> Help or by pressing F1 in the Settings window.

There are over 5300 topics in the forum so your question may already be covered. It is advised that you search for your issue first in the forum to see if it has already been answered before creating a post. If the forum search feature restricts certain words or isn't helpful, then kindly search the forum using Google or Bing as follows:


<enter search terms> site:classicshell.net/forum

e.g.1, if you don't know how to change Start button,

In Google or Bing's search box, type: change start button site:classicshell.net/forum

e.g. 2 if you want to know how to add some folder, let's say, 'Videos' to the Start menu, search like this:


In Google or Bing's search box, type: add videos start menu site:classicshell.net/forum


If your question is about the Start Menu, please state what style of the menu you are using and what Windows operating system your PC is running.


Searching the forum will save your time and our time too if you find a thread already with your query resolved. So please search before asking the same things which are already in the forum.


If you have trouble finding a setting, there is a search box in the Settings dialog where you can try typing its description. If you don't understand what a particular Classic Shell setting does, simply hover your mouse over it, and tooltip help will appear explaining what the setting controls.

Important posts in each subforum are also pinned at the top of each subforum for your convenience. There are Start Menu related guides and tutorials in the Start menu section of the forum. Explorer related tutorials are in its forum.

How to change the Start button: http://www.classicshell.net/forum/viewtopic.php?f=18&t=853
See here how to capture crash dumps if you are experiencing crashes or slowdowns: http://www.classicshell.net/forum/viewtopic.php?f=12&t=6
Information about translating Classic Shell can be found here: http://www.classicshell.net/forum/viewtopic.php?f=20&t=1344
How to create skins for the Classic Start Menu: http://classicshell.net/tutorials/skintutorial.html

When posting a new topic, please give it a clear and understandable title so that topic is easier to find when others search the forum.

When reporting a bug or starting any discussion, please report the version of Classic Shell you are using and also the Windows version you are using, and whether it's 32-bit or 64-bit. You can post in older threads if it is related to your issue. But for unrelated issues, try to avoid hijacking other users' threads by posting completely different questions in other users' topics.

If your issue is resolved, kindly mark the topic as solved by clicking the green check mark at the bottom of the post.

Author:  Gaurav [ Sun Dec 22, 2013 11:51 am ]
Post subject:  Re: IE add in state they are 32 bit
If you are using IE10 or IE11, 64-bit Internet Explorer processes are only enabled if Enhanced Protected Mode is turned on from Internet Options -> Advanced Tab. Also, on IE11 on Windows 8.1, the separate option "Enable 64-bit processes for EPM" must also be enabled in addition to the option "Enable Enhanced Protected Mode".

Author:  Gaurav [ Sun Dec 22, 2013 8:56 pm ]
Post subject:  Re: IE add in state they are 32 bit
You haven't stated whether you enabled Enhanced Protected Mode from IE Options -> Advanced tab and if 'Enable 64-bit processes for EPM' is also enabled. Only then on Windows 8/8.1, you will get all 64-bit IE processes.

Regardless of that, you don't need to worry about 32-bit or 64-bit. If 64-bit IE processes are enabled (by enabling Enhanced Protected Mode), the IE addons will still work. Just make sure "ClassicIEBHOClass" and "Classic IE Settings" are enabled.

Everything is simply superb, I use it a lot (have assigned Ctrl+Shift+F) to open its window. But it doesn't do content search, nor network shares. So I use 3 programs to search now depending on the type:
- Indexed stuff (Classic Start Menu which has Windows Search integration)
- Non-indexed file name only search (Everything)
- Non-indexed search with contents using IFilters (FileLocator Pro)
- Indexed search on the network (Add those shares to Libraries and then use Windows Search/Classic Start Menu)
- Non-indexed search on the network for contents (FileLocator Pro)

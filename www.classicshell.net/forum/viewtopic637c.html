Post subject:  Re: [Windows 10] Classic Explorer toolbar not present
If all addons are enabled in the Manage Addons dialog, and you also enabled the option "Enable third party browser extensions" and yet if you are unable to turn on the File Explorer toolbar, then it may be a case of File Explorer getting confused with the toolbars it has installed.

To fix that, first close all File Explorer windows. Then open Registry Editor (C:\Windows\system32\regedit.exe) and go to HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Toolbar\ShellBrowser key and delete the value called ITBar7Layout. Then re-open File Explorer and try enabling Classic Explorer Toolbar again.

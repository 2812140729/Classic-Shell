Post subject:  Annoying Taskbar
Hello.

Quite often, we move the mouse cursor down to the bottom of the screen because we need to select a slider or any object on a web site.
If you use the "hide taskbar" option on Windows sometimes it's difficult to work properly.
As you get close to the bottom of the screen the task bar grows up preventing you from clicking anywhere near.
Moving the taskbar to the top, righ or left side of the screen just moves the problem to a different area.


It would be great to have some solution for it.
In the meanwhile it could be a good idea to set a delay to the taskbar unfolding/unhiding action, or maybe allow some keyboard+mouse combination.
Another option could be to let the user to select elements through the taskbar as if it wasn't there, as if ot was transparent.

Author:  smashkenazi [ Mon Jul 03, 2017 2:37 am ]
Post subject:  Re: Annoying Taskbar
If we could change the width of the taskbar, it would solve some problems.

For example, I use remote desktop to login to a machine with a larger screen than my laptop.
I want more screen height, so I move the taskbar to the right instead of across the bottom.
Since the taskbar automatically extends across the full length, it now covers the top right corner of the window in which I'm working.

That means I can't hit the "x" or "_" to minimize or close the window.
Were the task bar length configurable, we could reduce the size slightly to stop this annoyance.

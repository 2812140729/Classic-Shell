I'm a beginner and just downloaded "Classic Shell" for my Windows 8.1 desktop. It seemed to me more attractive than "StartMenuReviver", the tool which I used before.

But after deleting the Reviver and installing "Classic Shell" I had to realize to my horror that the "Start" button on the left side of the taskbar disappeared. Only a tiny rest survived (and this rather atomic fragment is extremely difficult to find).

A rather temporary solution opened when I discovered the possibility to open the start menu by tapping the windows key. But untill now I do not find an approach to the settings of the "Classic Shell" program and to its different start designs.

Author:  Gaurav [ Sun Oct 16, 2016 11:20 am ]
Post subject:  Re: "Classic Shell" does not start
Using File Explorer, you can go to C:\Program Files\Classic Shell and open "Start Menu Settings". It will allow you to change the size of the Start button which you must've changed accidentally to some very small value.

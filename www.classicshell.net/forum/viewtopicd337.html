If an emblem is tied to the second column (using left2, center2 or right2), then it will disappear if there is no second column, like during search.
You can specify a secondary alignment for such cases. For example:
left2|right
This will make it left-aligned to the second column, or right-aligned if there is no second column. This is a 4.3.0 feature.

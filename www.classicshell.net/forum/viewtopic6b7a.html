Preferably any photo program that can zoom in enough to alter each pixel.
One that can work with, and export a transparent .png or transparent 32 bit alpha bitmap image.
Program needs to have a color picker, so exact pixel value can be set.
A pixel measurement ruler is handy too.

Practically any Adobe image editor should work. Paint.Net, Gimp, PaintShopPro.
Splitwirez has his Paint.Net. Jcee with Gimp. Ivo-Adobe, Gaurav -?, others-?

My favorite barebones editor is PixelFormer.
It can zoom in up to 6400%, making it easy to adjust the pixel line at top.
Adding an additional 1 pixel horizontal line should be easy in any program.

You cant necessarily add 'all' the bookmarks, but you can add individual ones via drag/drop
Just highlight the url in the address bar and drag directly into the start menu
Alternatively, if you want the link to be on the left side, you can create a custom menu item, and paste the link as the 'target' or 'link

Right click the start button, go to settings, Check 'show all settings' on the top, then Customize start menu

In windows 7 style:
Right click and select 'insert new item'
Click where it says 'administrative Tools' on your new link and select 'custom'
Double click on the 'icon' which should be a white 'page' at this point
Paste the URL into the 'link field'
Give it a name
Give it an Icon if applicable
Move it where in the menu you want it

In the classic/2 column styles:
Click on 'custom' in the right column(available commands) to add it to the left column (your menu)
Move it where in the menu you want it
Click it to edit
Paste the URL into the 'link field'
Give it a name
Give it an Icon if applicable

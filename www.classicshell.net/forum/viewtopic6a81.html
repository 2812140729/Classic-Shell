Post subject:  Windows 8 home screen permanent choice
Hi,

Thanks for your nice software. For me it provides almost all needed enhancements to Windows 8 GUI:
- Start menu
- Skips "Modern" start screen upon Windows login

However, one Windows 8 behavior that's really annoying me (and others from what I've seen on the web) is the fact that Windows always goes back to the mode of the last run application after closing that given application.
For example, if I launch and then close WordPad, Windows will always show the desktop screen after closing WordPad. If I launch one of the modern Windows 8 Apps like "Travel", it will always show the modern start screen after closing the modern app.

This is not really convenient for users like me who would prefer to always go back to a "current" mode. IE: if I launch any app from the desktop, I want to always go back to the desktop after closing the app. If I launch any app from the modern start screen, I want to be taken back to the modern start screen after closing the app. Basically, when I work in the modern GUI, I'd like to stay most often in this GUI (typically when using a touchscreen), but if I'm working from the desktop, I'd rather stay in classic desktop mode (typically when using mouse and keyboard).

Of course, this would not prevent users from accessing either the desktop or the modern start screen by clicking on their links or the windows key at any time.

So I was wondering if such feature would be something that could be part of Classic Shell?

p.s. Sorry for not posting in the "Feature request" section, but I can't seem to create a topic there.

Regards,

kik

Post subject:  Re: "Pin" the Start button to taskbar?
No; as that shouldn't be happening, though unchecking 'replace start button' MAY solve your issue, or trying various other options in that tab.
As a last option you could try creating a shortcut to: "C:\Program Files\Classic Shell\ClassicStartMenu.exe" -toggle" and placing it on the taskbar. It will act as a new start button.

Though we should get down to the real problem here, whats causing it
Try uninstalling classic shell, does this 'fix' the issue?
Try reinstalling the latest ver. does this 'fix' the issue?
Try reinstalling an older version to see if its a 'new' bug,does this 'fix' the issue?
What version of windows 10 are you on? (type 'winver' into the searchbox in the start menu)
Do you have any other Programs that tweak the taskbar or start menu? try disabling them?

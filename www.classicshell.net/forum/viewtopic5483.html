Post subject:  Re: Favorites in Navigation Pane?
Windows 7 by default should have Favorites at the top. There are some registry hacks to make it go away. Classic Shell doesn't do anything to add or remove that item.

Windows 8 has an option to toggle Favorites if you click on the "Navigation pane" button in the View tab of t he ribbon.

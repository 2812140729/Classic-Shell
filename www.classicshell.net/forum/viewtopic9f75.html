Here is the detailed change log for Classic Shell 4.3.1:

New features in Classic Shell 4.3.1:
● New command in the Backup button of the Settings window to clear the icon cache without restarting the menu
● Alt+Enter keyboard shortcut to open Properties of Start menu items and jumplist items

Other minor changes and fixes in Classic Shell 4.3.1:

● Fixed an issue with the Start button in some cases appearing in fullscreen programs and games.
● Added a confirmation dialog when you cancel the changes made in Settings
● The Settings window remembers its maximized state
● Fixed the bug where <No Skin> was missing icons
● Fixed a crash where the search would crash if the programs tree in Windows 7 style was disabled
● Fixed the broken taskbar texture for Windows 10 Creators Update
● Fixed a bug where sometimes the jumplist items wouldn't work
● Fixed the broken Start Screen shortcut for Windows 8
● Fixed a bug where the taskbar color was wrong on Windows 8 if only the opacity was customized
● Fixed a crash when dragging an item from a submenu or a jumplist to the search box to pin it
● Fixed a crash with Windows 10 Creators Update when pinning UWP apps from search results
● Updated code signing certificate for this version

Previous release was Classic Shell 4.3.0.

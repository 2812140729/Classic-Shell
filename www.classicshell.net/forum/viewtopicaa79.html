I forgot how good Gaurav's Skin was, especially on opaque setting.
Those registry tweaks made magic.

I'm still going to release my version of Ivo's Win 8 skin.
Code name "Mr Milky". World name "Win 8 alpha" [7 style]
This one works better with Windows auto color off.
For when it's on, it darkened menu too much.

I find it's really hard to make opacity,
menu brightness, and glass color match exactly to taskbar.
I switched to pure alpha transparency (more linear, but more milky look).

Settings
Windows color intensity on default
CSM Overide glass color off

or
Windows color intensity on default
CSM Overide glass color on
Menu glass color default
Menu glass intensity default
Menu color blending around 20







If windows auto color is turned on, these settings will brighten menu a little.
CSM Overide glass color on
Menu glass color default
Menu glass intensity default
Menu color blending around 65

Skin has an intermediate opacity setting of about 85 percent.
Set taskbar opacity to 85 to match.


notes
There is a bit of an oddity with setting taskbar opacity to 85.
If windows color is changed, taskbar still stays old color.
Just reset to opaque, and then back again.

Also skin is for Win 8.1, not Win 10.
Won't match in Win 10, because taskbar is darker.

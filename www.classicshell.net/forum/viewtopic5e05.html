You can try to use "align to the corner of the screen", it's an option in the button settings.

About dimensions, I don't know what program you're using to edit the image, but in most cases doesn't matter, it's very hard resize an image so much, I tend to build them from scratch and use the image as reference.

For the problem of the button floating, verify also you don't have some blank pixels on the side from where it's moving, maybe it's a question of alignment of the three images.

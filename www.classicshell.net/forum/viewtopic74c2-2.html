In Classic Explorer options (All Settings) -> Navigation pane tab, you can set the "Tree item spacing" to negative numbers (like -3 for example) but while it does slightly reduce the spacing between top level items from each others, it also reduces the item spacing between ALL items from each other, not *just* the spacing between top level items.

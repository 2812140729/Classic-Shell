Here ya go a clear version

also troy, here are the steps in gimp to accomplish this if your interested:
1. open image in gimp
2. select the 'fuzzy selection' tool (hotkey is U)
3. drag click on the background (while holding the mouse button down drag up or down to increase/decrease the selection sensitivity)
4. release when the area is properly selected
5. hold the shift key, and repeat steps 3-4, for the 4 background sections
6. now delete the selections (delete key)
7. save

Right click the image, save it somewhere it wont get deleted (I recommend creating a folder labeled 'Start buttons' in classic shells install directory, though it doesnt really matter where)
Then right click your start button, and from the start menu style tab, check "Replace start button" select custom, then pick image. Then browse to the file and hit OK

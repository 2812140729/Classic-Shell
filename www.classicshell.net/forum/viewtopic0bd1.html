Post subject:  Re: Help Needed Windows Classic shell Please ?
What kind of help do you need? If you installed Classic Shell, it will ask you for the style of the Start Menu - click on the image of the style you want to use and click OK and you can use the Start Menu right away. To access any settings again, you can right click the Start button.

Every setting is explained by tooltip help. Hover over a setting to see the help. On the Customize Start Menu tab, you can change what items show in the right column of the Start Menu. The rest of the settings are quite advanced, you don't necessarily have to set them.

To search for a setting related to the Start Menu, open its settings and use the search box at the top of the settings window.

To search for Windows settings, type directly in the Start Menu search box.

When you install Classic Shell, you will get a dialog like this:

Click Windows 7 style there and then click "Select skin...". Then pick "Windows Aero" from the drop down and click OK.

This is as Windows 7-like as you can get it. However, the user interface of Windows 10 has changed a lot from Windows 7. You can't make it like Windows 7. Many settings are moved to the Settings app and the apps and notification area/tray applets are completely different.

Author:  Gaurav [ Tue Sep 26, 2017 10:55 am ]
Post subject:  Re: Option to run CS without resident ClassicStartMenu.exe
@Splitwirez, You can already have Classic Shell not autostart when Explorer starts by unchecking "Start automatically for this user" on General Behavior tab. Run it manually from C:\Program Files\Classic Shell\ClassicStartMenu.exe.

Of course, once it starts, the ClassicStartMenu.exe process will run consuming negligible amount of memory. It only keeps running to automatically launch the menu *if* Explorer restarts or crashes.

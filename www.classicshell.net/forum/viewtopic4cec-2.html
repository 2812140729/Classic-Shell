EOSX (Classic 1 and 2 column) Use with minimum CSM version 4.2.6
Does not work at all with v4.2.5

Could not transfer all the features of this old "Eclipse OSX" menu, to Classic shell skin.
So I renamed it EOSX. No, not OSXE Capitan. That's too new.
This is from an XP Msstyles group named "Community".
It looks like it was made by "TGT XP Visual Style", back in 2001. Their web site looks to be taken over by online casino info.
Too bad, they made some great styles.

I modified it quite a bit. So I guess the EOSX mod, could stand for Engineering Ordered Surgical Xtraction mod.
The way I fudged some of the slices, is not the correct way of doing it. It's working for now anyway.

Could not get texture on submenu, so put on main menu instead.
Added soft glass buttons option.

Option for extending bottom text paddings onto bottom border.
Last of all, option to selectivly cut off the top, bottom, or both borders, for users with taskbar at top of screen.
Personally, I keep the rounded corner border visible, as that's the best part of their menu design.


Thankyou TGT, for the bitmaps screenshots, keeping them alive.

I'm sure you can find a better trademarked Apple start button, than the screenshot mod included.
I must've had one hundred text logic mistakes at one point, because I kept adding options, that messed up everything from before.
Most probably fixed them all now, fingers crossed.

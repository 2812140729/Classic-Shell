Yeah lots of users are reporting this. Over time, it just stops skipping the Start screen. Best solution is combination of UserInit and service, as Task Scheduler will also start it delayed even if you reduce the startup delay using this reg tweak: http://winaero.com/blog/how-to-reduce-t ... windows-8/ . Since installing a service may also not be acceptable to some users, it should in fact be an option during installation, which if unchecked, adds Classic Start Menu only to the UserInit key.

Classic Shell uses icons from Windows DLLs because they cannot be redistributed as part of the Classic Shell product. The ones that are missing don't have icons in Windows DLLs.

You can get some nice icons from websites like www.iconfinder.com. Some of them are freely licensed and do not even require a backlink/credits.
If you are a fan of Windows XP style icons, some design hobbyist recreated them in 256 x 256 size on Deviantart!:

Get the XP style icons here:

https://www.dropbox.com/s/f066o8j2m1vc2n7/Windows%20XP%20Large%20Icons.rar?dl=0

Get the Vista/Windows 7 style icons here for some toolbar commands and some Start Menu items:

https://www.dropbox.com/s/ozkb9av08hnol5n/Icons.zip


There are also websites like iconfinder.com, iconarchive.com to find freely licensed, not copyrighted, high-resolution icons.

On Windows 10 the icon for "Sort by" is "%SystemRoot%\System32\imageres.dll, 5312", the icon for "Group by" is "shell32.dll, 16748", and the icon for "Properties" should actually be "%SystemRoot%\System32\imageres.dll, 5367" not "shell32.dll,253".

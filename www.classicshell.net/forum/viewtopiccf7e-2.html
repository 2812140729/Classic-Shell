Yes, the start button means the current start button - either the Classic Shell button or the default Windows 10 button. The uninstaller will attempt to close the start menu, but if it doesn't, you can help it by exiting the start menu yourself.

The toolbar instructions are mostly for Windows 7. Windows 10 has a ribbon, so the menu is not that important.

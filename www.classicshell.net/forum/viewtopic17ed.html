http://sourceforge.net/projects/softwarepolicy/

Basically a less intrusive replacement for user account elevation, no screen dimming, no aggro when changing control panel settings, and no problems with older non UAE-aware programs provided they follow Windows' guidelines as to where files should be stored. (which most do)

Uses software policies to restrict the places on disk where programs can be run from. With the policy in force, most installers will fail rightaway, keeping junk out. Optionally, running of programs from CD or USB memory can be blocked. To install wanted software, you unlock from a system tray icon. This gives you 30min of 'superuser time' with no more nags, after which it relocks automatically.

Runs selected processes (typically browsers such as Firefox/Chrome) as a restricted user, but with same settings and preferences as the regular account. Provided you have standard folder permissions this prevents the webpage from dropping files into system folders or various other malicious actions.

It is intended to be run with UAE off, you can run with UAE on as well for extra security, although in this case you will see a UAE prompt asking to allow SoftwarePolicy.exe to run at each startup.

Not yet tested on Windows 8.1 (will be soon)

PortableApps
http://portableapps.com/


How about having a Windows OS that is completely Install free?

None of the Programs that work with PortableApps can be installed on your computer, they are only portable versions of well-known Programs, ported specifically to PortableApps

It also has its own Start Menu

This means that you can take your entire OS with you on a USB Key, and have it anywhere you plug it in... all the apps and programs and data intact, provided you save it all onto your same USB Flashdrive.

It also means you don't have to worry about all the security problems, all of the malware, or all of the problems that go along with full-installs.

I use this for running around to different computers in different offices, businesses or classrooms when giving lectures or training. It's like having your entire computer in a USB key. You can use everything you need, without leaving a footprint or needing to install anything that a computer you use may not have or need special permissions to access. Works great at libraries or public computers that don't give you access to anything.

I suppose it could even be made to work with WinPE, which would mean you could use it anywhere without an OS even.

Portableapps works best if you also keep all of your other portable versions of your favorite programs on the same USB key, so that you can use anything that is portable that you need, in case PortableApps itself doesn't have it available for download.

X-mouse Button control
Freeware
https://www.highrez.co.uk/downloads/xmousebuttoncontrol.htm

This handy program allows you to assign custom functions to your mouse based on what program is active, For example I have assigned win+scroll Up to increase the Volume. globally and have reconfigured the side buttons on my mouse to move between tabs in chrome. Configuration is a little tough, but once you get it configured Its quite useful.


StrokesPlus
Freeware
http://www.strokesplus.com/
Adds custom mouse gestures to windows. This software is highly customizable and can do anything from sending a hotkey when a gesture is made, to opening a specific website or program. It compliments X-mouse quite well, and allows an unlimited number of functions to supplement the few buttons available on a mouse. I would strongly suggest changing the default button to either middle click, or one of the side buttons on your mouse. (right click is likely to interfere with alot of games/software)

Chrome extensions:
Since many people do use google chrome, i thought I'd add a post or 2 about the everyday extensions that are OFTEN useful.

Adblock - goes without saying..., though some ads have found a way to skirt through in recent years (and some sites go on lock-down until you disable it)
Fresh Start - session manager.. This extension saves all (or whichever tabs you select) as a session (actually it bookmarks everything, in a hidden folder, so it works across computers, or system reinstalls) It also has a crash recovery function, that can restore windows/tabs closed a day or 2 ago (which is very convenient when chrome occasionally looses tabs
Lazarus form recovery - This extension stores form data, (like really long forum post) so if you accidentally hit the back button, or close the tab. all that precious text isnt lost.
Clutter free - this extension ensures you dont open duplicate tabs. If your workflow is like me (with a hundred open tabs at a time) its quite easy to do.
PageArchiver - this extension requires a parent extension be installed on your computer, but it effectively converts webpages into single-file documents for offline viewing (and archival purposes)

Post subject:  Re: Microsoft Windows 10 (B1607) Anniversary Update Clock?
You can get old clock layout using the registry editor.

Press Win+R and enter "regedit" and hit enter
Open the following key :
HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\ImmersiveShell
Click on the Immersiveshell folder on your left hand side. Following that, create a new DWORD (32-bit) Value in your right hand side. Name it UseWin32TrayClockExperience. By default, the value would be 0. You have to set the value to 1. To change the value, double-click on UseWin32TrayClockExperience and enter 1 before saving your change.

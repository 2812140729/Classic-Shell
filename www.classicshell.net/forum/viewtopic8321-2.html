Post subject:  Re: different Buttons for new folder with standard name
I assume that by "short cut bar" you mean the Explorer toolbar. Open Classic Explorer Settings and go to the Toolbar Buttons tab.

Scroll down in the right column until you locate the last item "Custom". Create a custom command for the Explorer toolbar by dragging "Custom" to the left pane. Position it where you want and then double click it to bring up the Edit Toolbar Button dialog. In the Command: field, specify this:

cmd /c md "%1\Request"

Click OK to save changes.

Do the same for the other command:

cmd /c md "%1\Quotations"

Post subject:  Re: Desktop F3 Keyboard Shortcut
No replacing F3 is not a good idea because F3 opens different things on Windows 7, 8.1 and 10 and users prefer different things. For example, on Windows 8.1, the built-in Windows search is the only one that can search network files, Classic Start Menu cannot so both are required. Similarly, on Windows 10, some users prefer Cortana whose functions Classic Start Menu cannot entirely replace.

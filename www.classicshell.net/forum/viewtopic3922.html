If by 'top caption' you mean the user name text, that parameter is called "User_font" in the Skin file. I changed it to User_font="Segoe UI",normal,16 instead of the default Segoe UI",bold,22. Makes it fit longer names. Make sure you open Resource Hacker as admin if you are directly opening and saving the skin file from C:\Program Files\Classic Shell\Skins\.

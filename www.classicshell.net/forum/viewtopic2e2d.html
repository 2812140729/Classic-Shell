Post subject:  Re: Make start at bottom in vertical taskbar
There is currently no easy way to do it. but if you're willing to put in an hour or 2, you could make a 'fake' start button with rainmeter, and on click have it run "C:\Program Files\Classic Shell\ClassicStartMenu.exe" -toggle
The Tray/volume/clock/and Notifications will still be on the bottom. (and the bottom-most element would be covered up by your fake button)

This menu would open from the top down, but if you had enough menu items to fill the screen it would appear the same. If you don't want that many items, you can further complicate things by adding padding to the top of one of Classic Shells .skin files to push the menu down to where you need it. I would estimate it at about 1.5 hours (with no prior rainmeter experience) and another 30 minutes if you wanted to figure out how to pad a skin.

If you decide you want to go this route, I would be happy to give more detailed instructions, and help you, should you have any questions. though I suspect you are looking for a simple, solution. which is up to IVO

Post subject:  Re: Make start at bottom in vertical taskbar
Tricking the resolution wont work, nor will making a really tall start button, with the bottom being the only visible part (it will just push down all the other buttons)
as for responsiveness..

I actually decided to try it out for myself, and the click appears responsive enough, however there is some Z-position issues that make the start button go behind the taskbar for 1/4 a second after the button is clicked (if you make a custom taskbar skin, and make that section clear, the effect will be minimal)

Heres the how-to, and the new eta is 2-3 minutes (since I did all the leg work))

After installing rainmeter:
Take this file, unzip it to C:\Users\USERNAME\Documents\Rainmeter\Skins\Chrome\
then right click on rainmeters taskbar icon, go to 'skins' find chrome.ini
now drag the icon to the bottom corner, right click, and change its properties so it can't be dragged anymore.


If this meets your needs, you can modify chrome.png and chrome2.png in the zip for more appropriate icons..

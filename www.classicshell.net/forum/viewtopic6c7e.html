Author:  Ivo [ Tue May 02, 2017 7:03 am ]
Post subject:  Re: slow start menu on Windows Server 2012 R2 Terminal Serve
It is trying to read the jumplist contents from this folder: %APPDATA%\Microsoft\Windows\Recent\AutomaticDestinations. It takes a few seconds to access it in your case.
You can turn off the jumplists from the start menu settings if you want to sacrifice that feature for performance.

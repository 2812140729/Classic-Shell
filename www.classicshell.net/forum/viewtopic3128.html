Post subject:  Re: Bring Back Windows XP Skin In Classic Shell 4-Suggestion  
Classic Shell 4 has not removed the Windows XP skin but moved it elsewhere. To get back the Windows XP skin, first go to the Style tab in Start Menu Settings and choose the "Classic with two columns" style, then go to the Skin tab and select 'Windows XP Luna' from the Skin tab. Note that the XP Luna skin is not available for the new Windows 7 style of the menu. You must click "Classic with two columns" style to use it.

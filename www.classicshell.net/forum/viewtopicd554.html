Post subject:  Re: No Buttons in bar
There are multiple Explorer addons. Make sure all of them are enabled:

For Windows Explorer:
- Classic Explorer Bar
- Explorer BHO Class

For Internet Explorer:
- ClassicIE BHO Class
- Classic IE Settings

The toolbar is usually automatically enabled, but in case it isn't, to enable the toolbar in File Explorer, you may need to right click the menu bar (on Windows 7) and then enable "Classic Explorer Bar". On Windows 8, you will need to go to the View tab of the Ribbon and click the little arrow on the Options button and enable Classic Explorer bar.

Author:  Gaurav [ Mon Nov 18, 2013 5:27 am ]
Post subject:  Re: No Buttons in bar
The toolbar you see below the search box that says "Search This PC" is the Classic Explorer toolbar with buttons for Cut, Copy, Paste, Delete, Properties, Email and Classic Explorer settings. Since the Ribbon above it takes a vast amount of space, you can minimize it by pressing Ctrl+F1. Then you can click the button for Classic Explorer settings and add any commands you want. Check the box in Classic Explorer settings called "Show All Settings" and then go to the Toolbar Buttons tab. On the Toolbar Buttons tab, the right column lists all the available commands and the left one is where you must drag them from the right to add the ones you want. Once you drag the command to the left, you can double click it in the left column to edit the label (name), command, tip etc. If you don't want the labels to show, keep the label field empty.

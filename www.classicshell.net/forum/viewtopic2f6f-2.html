Author:  Gaurav [ Sun Apr 19, 2015 12:02 pm ]
Post subject:  Re: Can't Locate "Claude Glass Skin"
It was a very old classic style skin that I created in 2010 based on the Smoked Glass skin. It was OK for 2010 but now it's kind of 'meh' I didn't maintain it after that. Here it is attached. Note that this skin is very old and doesn't have MANY new features that newer Classic Shell skins incorporate such as highlighting new programs, high DPI awareness, split buttons, Aero auto colorization, support for 'no icons in second column', localized tooltips, numerous bug fixes and the expanded skin format with Classic Shell 4.
So while Classic Shell has great backward compatibility with older skins, and this old skin should continue to work, many of the Start Menu features that I mentioned above won't work. Use it as it is. It is unsupported.


Feel free to modify it as you want.

Rather, than doing that. (though it is possible, im somewhat busy as of late)
Here are the steps to change the font manually;
Download Reshacker http://www.angusj.com/resourcehacker/#download
Open the .skin file with reshacker
Click the skin folder, then the only file in it
then Ctrl+f to open a search
type 'font' then change the font listed
hit F3 to find the next instance of the word 'font' (this may only work in v 3.60 i recall the new version having this feature broken)
then replace the font type listed on the same line again
repeat till there are all fonts are changed, then compile, and save (you gotta save on desktop or downloads, because reshacker does not have access to save in your skins directory)

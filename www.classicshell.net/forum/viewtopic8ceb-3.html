Post subject:  Re: Glass effect in "all programs" menu w/ the Win7 start me
It is definitely possible as long as you are using the cascading menu in Windows 7 style (the background must be white for the inline treeview but cascading menu can definitely have glass). To add glass to the submenus, make sure you add Submenu_opacity=fullglass, the correct bitmap with transparent and the mask bitmap if required, and all the slices are correct. You could copy-paste some parameters from some skin for the classic style that already has glass for submenus and then modify them to suit your skin.

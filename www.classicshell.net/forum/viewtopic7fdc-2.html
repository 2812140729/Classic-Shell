Yes it's possible. You can do that from Classic Explorer settings -> Backup button. Are you having a problem importing or exporting the XML file? You will need to copy over any icons and commands you specified manually. If you use environment variables in their paths, the same XML will work on most machines. Classic Shell automatically uses environment variables wherever possible.

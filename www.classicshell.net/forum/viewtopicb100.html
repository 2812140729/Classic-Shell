Author:  Gaurav [ Fri Oct 20, 2017 7:29 am ]
Post subject:  Re: Classic Start Menu opens and closes after Session Logon
If the remote computer has many tasks set to run at logon or startup, I think it's normal if some of them are stealing focus as you just log in to your session. Just wait for a few seconds.

If you can identify which apps or tasks are doing this and they are not necessary, you can disable them. SysInternals AutoRuns would be the best tool to show you everything in detail: https://docs.microsoft.com/en-us/sysint ... s/autoruns

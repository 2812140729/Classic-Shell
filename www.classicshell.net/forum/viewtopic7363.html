Post subject:  Option to Hide Compressed files in Explorer Navigation Pane
Please add an option to Classic Explorer to remove compressed files from the Navigation Pane.

Although I like that windows explorer can open compressed files and show their contents, I don't like compressed files showing up in the navigation pane since it makes navigating folders that contain both a larger number of sub folders and compressed files more confusing and difficult than it was in previous versions of windows, particularly if a compressed file has been extracted to the same folder containing the file since the compressed file and the base of the extracted folder structure will often have similar names.

It also may be helpful to have an option to only show the compressed file and folder structure in the navigation pane when the compressed file is open in the details pane.

Hello,

the Windows 7 style start menu, which can be selected in the Classic Start Menu Settings, currently behaves exactly like the original Windows 7 start menu, meaning, you always have to click "All Programs" to see all programs.

Could you please add an option to the Classic Start Menu Settings which makes the Windows 7 style start menu show "All Programs" by default, so that a user doesn't always have to click "All Programs" to see all programs?

It would be much appreciated.

Author:  Gaurav [ Sun Feb 16, 2014 12:46 am ]
Post subject:  Re: Windows 7 style start menu default to "All Programs" opt  
This feature is already present. Right click the Start button -> Settings. Check 'Show all Settings' at the top and go to the Main Menu tab. There is an option "Open automatically" under "All Programs style". Check it and click OK.

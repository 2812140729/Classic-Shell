Post subject:  Skin: Aero Glass  
Aero Glass, is a modified version of smoked glass featuring modified background texture and adjustments to the username and picture elements.


Along with matching the color of your Taskbar/Windows Theme, this skin currently has the following Options:

-Two Columns
- -No Icons in Second Column
-Show User Picture
- - Black Picture Frame
-Show User Name
- -Center Name
-Small Icons
-Taskbar on Top
-Dark Side Column
-Add Glass Color
-Increase Transparency
-Black Font
-Custom Image

And Variations:
-Dark
-Gray
-Light

UPDATE Mar 31th:

- Removed the custom background image (all the framework is still there, you just have to remove the ; from the skin option, and from the settings at the bottom of the first skin resource. and import an image as 31. may still break away with a new version.
- Added a bitmap mask image, so certain parts of the image (its edges, and the lines) are more distinct when a custom color is added.


Please post any bugs, suggestions and request below.

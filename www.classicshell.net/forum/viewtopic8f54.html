Post subject:  Re: Upgrade check?
If a setting is manually modified (shown in bold), then it will preserve its value. If a setting is not bold then it will use whatever the default value is. The default value is something that depends on various factors, like OS version, DPI settings, etc. Also sometimes it changes between Classic Shell versions because I chose to do it. I don't do this often, and I'm doing it during the beta more often than during regular releases.

Some features may work fine between them with the right settings. and it shouldn't hurt your system to try (there should be no lasting ill effects after one or the other is uninstalled)
Just note the menu's themselves wont somehow 'merge' but you can probably set it up where one hotkey opens start10 and another classic shell. (or clicking the start button for classic shell, and windows key for start10 ect)

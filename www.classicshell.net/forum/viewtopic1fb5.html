So apparently Microsoft messed up, yet again... and broke the upgrade assistant tool... (It Requires a DLL that isn't present, and cant be installed on windows 7)
The solution:
https://www.addictivetips.com/windows-tips/fix-the-api-ms-win-core-libraryloader-l1-1-1-dll-is-missing-error-windows-10-upgrade/

(TLDR copy C:\Windows\System32\wimgapi.dll toC:\Windows10Upgrade\wimgapi.dll replacing the old file)



If you are on windows 7, 8, or 8.1 and want to upgrade (which you can still do for free (till Dec 31st) if you ever used the onscreen keyboard, narrator, or the magnifying tool)
https://www.microsoft.com/en-us/accessibility/windows10upgrade?tduid=(14c348e893a91089cb2980533c6ee123)(256380)(2459594)(TnL5HPStwNw-0FBZfNm6YPFXNhiu3o52rA)()

Though, if you are comfortable with Windows 7, I wouldn't recommend the 'upgrade' (though you can always try it and revert later if you dislike it)


Just thought I would post, as it took me a good hour or so to find this fix, and it doesn't look like Microsoft will be patching it anytime soon.

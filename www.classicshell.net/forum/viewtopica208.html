Post subject:  Re: Looking for Useful Custom Explorer Toolbar Buttons
Some buttons on the Ribbon are not yet available for the Classic Explorer Toolbar. These include:

- Copy Path
- Pin to Quick Access (Windows 10 / Add to Favorites (Windows 8/Windows 7)
- Open Command Prompt at currently open folder / Cmd as Admin
- Open PowerShell at currently open folder /PSH as Admin

If you install StExBar (another EXCELLENT free Explorer toolbar), you get buttons for "Copy Paths", "Copy Names", "Open cmd here", "Open cmd here as admin", an outstanding "Filter" box for Explorer and a regular expressions-powered Rename tool.

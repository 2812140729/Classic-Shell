Matthews wrote:
LOOKS AMAZING !!!
How can I download or install this?

What folder do I download it to so I can use this as my start menu button?

You just right click the image and choose "Save picture as..." or "Save image as..." in your browser. Save it to any folder you want e.g. A folder called Start Buttons on your Desktop or inside Documents. Then right click the Start button -> Settings. Check the "Show all settings" checkbox at the top and go to the Start Menu Style tab. Pick a "Custom" button there and click Pick image and then Browse to the image that you just saved and click Open.

If you want to adjust the size of the button, go to the "Start Button" tab where you can fine tune more settings related to the Custom button.

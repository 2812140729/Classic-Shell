I am sorry, but surprisingly I did not find a topic for the German translation ... though I know Classic Shell is being used in Germany quite a lot.

So, I have a translation change requstion.

In General Behaviour it says
in English: Start automatically for this user
in German: Beim Starten diesen Benutzer automatisch anmelden

Translated back to English it would mean "Automatically login THIS USER when starting [Classic Shell]" which makes no sense.

The fix is actually easy by just changing it to "Beim Starten dieses Benutzers automatisch anmelden" which changes the context completely and makes it right

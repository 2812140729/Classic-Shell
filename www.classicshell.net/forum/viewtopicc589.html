Post subject:  Re: Start Menu appearance changed overnight
Im guessing one of your settings was somehow changed (small icons comes to mind, but since I cant see the screenshot I cant confirm)
Lets try resetting everything to see if this solves the issue

First lets make a backup of your settings:
Right click your start button go to >settings
then click Backup near the bottom, save your current XML
Save this file somewhere

Next lets load the defaults
Again click Backup
This time click 'reset all settings'
Now hit OK and try the menu out
If this resolves the issue, than it is one of the settings that was changed.
You can re-load your XML and try to find which setting is causing the issue (Note: any setting that isnt set to its default values will be BOLD)

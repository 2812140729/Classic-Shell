Post subject:  Re: пропала кнопка «вверх» на панели проводника.
This error means "access denied". Unfortunately it is hard to know what is failing.
Choose Skip. If this is the only failed module, the only thing you are going to miss is the ability to Pin programs from inside 32-bit file browsers.
You can also try manually registering the DLL after the installation completes. Use this command from an administrative command prompt:
Code:
regsvr32 C:\Windows\SysWOW64\StartMenuHelper32.dll

Post subject:  "Up" Button Has Disappeared ... !  
Hello everyone. I've been a satisfied user of Classic Shell for about 8-9 months now, but recently an issue has arisen. When I first installed Classic Shell, everything worked exactly as it should've, and I was very happy. Recently, however, the "Up" button, which I prize greatly, has disappeared on me in my Explorer windows! I went into Classic Shell, and the options are all still properly set, but no Up button. I tried disabling the Up button then re-enabling it, but it didn't work. I tried moving the Up button to before the Back/Forward buttons, but that didn't help either. I even tried downloading the most recent version of Classic Shell and installing that, but to no avail. I also tried un-installing Classic Shell and then reinstalling it, but still no good. The Up button just isn't there. I'm using Windows 7 with Service Pack 1 installed. This is really frustrating, especially since Classic Shell worked perfectly for so long before suddenly deciding to take away my Up button! Can anyone help me, please? Has anyone else had this issue too? Any help on this matter would be greatly appreciated.
Author:  Gaurav [ Mon Jun 09, 2014 7:16 am ]
Post subject:  Re: "Up" Button Has Disappeared ... !
Sometimes IE disables Explorer's addons because the addon model is shared between the two. Try following the steps here: http://www.classicshell.net/faq/#explorer_toolbar
Author:  Hal_Jordan [ Mon Jun 09, 2014 2:03 pm ]
Post subject:  Re: "Up" Button Has Disappeared ... !
It worked! I just had to open IE and go to Tools --> Manage Add-Ons, and sure enough, Classic Shell had been disabled. All I had to do was enable it, and *ta-da!* all of the functions started working again (including the "Up" Button, thank goodness, lol).
Thanks so much for your advice!

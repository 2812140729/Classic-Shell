Post subject:  Re: Reset button to set all options to Windows 7/8 default o
There is a "Backup" button (to the left of OK/Cancel buttons) on every tab of the Settings of any component from where you can reset all settings to their defaults. To reset individual settings, right click them and select the option to reset it. If you want to go back to Windows defaults, uninstall Classic Shell or remove individual components. You can also disable the Classic Explorer and Classic IE components from Internet Options -> Programs tab -> Manage addons and disabling the addons there.

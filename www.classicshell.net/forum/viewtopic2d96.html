It is probably trying to access a network location and waiting for a response.
Possibly related to the user profile being on a remote machine.

In this case the menu is trying to get the timestamp for a shortcut in the start menu to see if it has changed. That shortcut is probably on a remote machine.
See if disabling the settings "Highlight newly installed programs" makes a difference for you.

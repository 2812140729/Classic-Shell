Post subject:  Re: Are there any more styles for download?
The forum you are in: viewforum.php?f=17 has skins. Styles are what the menu has pre-defined - Classic single column, Classic with two columns and Windows 7 style. Each style has skins. Skins are downloadable and can change the look of the menu, skins are specific to each style, so you will have to switch to the correct style to use the skin from that style.

If the extension is .skin, the skin is for Classic style.
If the extension is .skin7, the skin is for Windows 7 style.

Each post in this subforum: viewforum.php?f=17 has one skin. The skin file you download must be put in C:\Program Files\Classic Shell\Skins and then you must switch to the correct style: Windows 7 style or Classic to use it. After setting the style, go to the Skin tab in Start Menu settings.

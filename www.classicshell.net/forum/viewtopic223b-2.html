Author:  Ivo [ Mon Feb 25, 2013 12:12 pm ]
Post subject:  Re: Show icon in title bar always active
By default on Vista and Windows 7 when using the Aero or Basic theme, the text and icon are hidden in the title of the Explorer window (but never on the taskbar). In all other cases (when using Classic theme, or when running Windows 8), the title always shows the icon and text.

The Classic Shell option forces the icon and text to be shown only in the first case - (Vista/Win7, non-classic theme). In all other cases it does nothing.
There is no way to hide the title in Classic theme, or the taskbar.

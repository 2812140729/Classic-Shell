Also, if you don't want Frequent programs to be empty, you must turn on the Windows setting for it from Taskbar Properties -> Jumplists tab -> Store and display recently opened programs (Note that Windows calls this "Recently opened" but in reality, they're most frequently used programs that Windows keeps a list of).

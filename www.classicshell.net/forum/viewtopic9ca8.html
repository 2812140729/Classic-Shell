Actually it depends on the style.

For Classic styles you have to edit the command for the ShutdownBoxItem in the Customize Start Menu tab.

For Windows 7 style the setting is named "Shutdown command" near the bottom of the Main Menu tab.

Double-click on the ShutdownBoxItem near the bottom of the "Current menu items" in the Customize Start Menu tab.

Change the command from shutdown_box to shutdown. Or to restart. Or to whatever you want.

The left pane of Explorer uses the standard treeview control whose behavior can be modified because it is documented. The right pane of Windows 7/8 Explorer uses an undocumented private ItemsView control. A registry hack or a tool like Folder Options X: http://free-sk.t-com.hr/T800soft/softwa ... ptions.htm can force the ListView control for the right pane which is documented/standard.

I don't know however if the selection color can be changed in ListView because Aero-based themes use visual styles/bitmaps. If you change to "Windows Classic" theme, you will get a darker selection color which can also be changed in Personalization Control Panel -> Window color -> Advanced appearance. Under "Items", set the colors for "Selected Items".

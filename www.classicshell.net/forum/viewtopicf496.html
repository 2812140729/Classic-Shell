Temporarily disable the Classic Explorer addon. Go to Internet Options -> Programs tab -> Manage Add-ons. Disable Classic Explorer Bar and ExplorerBHO Class. Reopen the Explorer window and see if you can adjust the navigation pane spacing. If not, then your problem is not caused by Classic Shell.

If you can adjust it after disabling Classic Explorer, re-enable those addons and open Classic Explorer Settings. Use the big "Backup" button to save your settings to XML and upload/attach them here.

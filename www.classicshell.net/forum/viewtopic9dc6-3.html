Just thought I would report what may be a tiny bug...
I have Classic Shell Start Menu set at default - just installed in Windows 8 64 bit.
In the
"Settings for Classic Start Menu 3.6.5", "Windows 8 Settings" tab, the option to "Disable active corners" for the "Start screen" is checked.
This works correctly for a horizontal taskbar on the bottom.
But with a taskbar on the left, right or top, the Start screen active corner still shows up when hovering over the lower left corner.

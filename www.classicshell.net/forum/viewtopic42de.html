Hey Guys, Here's the new skin named Omni-Pic.
It presents a super minimalistic experience with tons of options to customize it according to your taste.
You can choose different background images to make menu prettier.
Select Different colors for text and submenu backgrounds.

System Requirements:-
1] Windows 7 or Higher
2] Classic Shell


How To use the skin:-

1] Download The Zip File And Then Extract It.
2] After that, Copy Omni-pic.skin7 to the Folder at: C:\Program Files\Classic Shell\Skins. Select OK to any User Account Prompt.
3] Select the Windows 7 Style In Classic menu Settings and load the skin in the skin tab.
4] Load the XML file for better experience. Button is located near the bottom for that.
That's all.

Special Thanks to Juniper7 And Gaurav for their continuous guidance and support.

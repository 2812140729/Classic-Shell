This is my first skin I've made and my first post here.

Based from Metro skin, I tried to modify Smoked Glass with Metro style and more transparent glass, so I named it "Metro Glass".
In this skin, I decided to add icon frames not only in large icons menu, but also can applied on second column, small icons menu, and even submenus. I also add the "opaque dark submenus" option because I want to match it with new windows 10 dark context menu.

NOTE: This skin isn't following the start screen colors, but for Windows 10 users can match it with accent color by unticking the "Override glass color" option in Menu look tab

If you have any suggestion or found any lacks, please tell me.
For Classic Shell 4.2.1 and up

To get the look like in screenshot, open CSM settings and:
- Set large icon size=32, small icon size=16
- Tick the "override glass color" option and select any color you want
- Set "Glass opacity" to 20
- In the skin options, tick the following options: No icon in 2nd column, Icon frames on main column, Icon frames on submenus, Show user picture, and Reduce glass color.

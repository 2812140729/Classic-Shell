Post subject:  Classic Shell 4.1.0 is released
Here’s what’s new:
  • Improved support for Windows 8.1 Spring Update
    • You can pin and unpin apps to the taskbar from the start menu
    • The start button scrolls together with the taskbar when they show on the start screen
  • Improved checks for new versions
  • Minor stability fixes

If you are currently using version 4.0.4, 4.0.5 or 4.0.6 you may not get a notification about the new version. It is recommended that you upgrade to the latest version manually.

The new version can be downloaded from the home page http://www.classicshell.net


Next release is Classic Shell 4.2.0 beta.
Previous release was Classic Shell 4.0.6.

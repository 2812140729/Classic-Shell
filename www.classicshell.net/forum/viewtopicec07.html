Classic Shell is like a regular app so it is highly unlikely to prevent an entire upgrade of a Windows version. The Classic Start Menu is an app, the Explorer extensions also use APIs. So as such it's not a hack or anything that modifies system files or settings and would cause the OS upgrade to fail. You can be sure of that.

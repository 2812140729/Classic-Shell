It is working on windows 10, but you won't like the fix I found. I went to a Windows 7 Ultimate 64 bit laptop that had Classic Shell 4.3.0 running perfectly and created two backup files. I loaded the main backup into the windows 10 anniversary classic shell 4.3.0. It didn't work at all. The settings file from the Window 7 machine did not create the same settings on the windows 10 machine. On windows 7 the left click opened the classic shell menu and on windows 10 it was set to open the windows start menu.

The original install of Classic Shell on windows 10 anniversary had both start buttons active. With the upload of the windows 7 menu file into windows 10, only the Classic Shell button is active. It seems the windows 10 64 bit install is different than the windows 7 64 bit install and the backup files do not align.

It is working perfectly on the Windows 10 machine now.

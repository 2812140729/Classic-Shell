Post subject:  Run as Different user
I currently have two machines running Windows 10 - one with the default Start Menu and one using Classic Shell. So far, I have been very happy with Classic Shell, with one minor exception - I do not have the ability to right-click on a program and tell it to run as a different user.

Please not, this is not the same as running as administrator. I have some cases where I need to run a program as an entirely different user than the one currently logged in. I am able to do this with Windows 10 but not with Classic Shell.

Would it be possible to add this feature to Classic Shell?

Detail on this feature can be found at http://windowsitpro.com/windows-10/addi ... windows-10.

Thanks,

Steve
Author:  Ivo [ Tue Sep 06, 2016 2:28 pm ]
Post subject:  Re: Run as Different user
Hold down Shift when you right-click. You will get the extended context menu with more options.
Note: this is not a Classic Shell feature - the extended menu is a feature of Windows.

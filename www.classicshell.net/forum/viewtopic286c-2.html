Author:  Gaurav [ Sun Jan 13, 2013 7:54 am ]
Post subject:  Re: Start button does not work
What do you mean Start button doesn't work at all? You don't see it or clicking it takes to Metro? Did you clean install Windows 8 or upgraded from Windows 7? Somehow it looks like your Start button got turned off. By opening the menu using Ctrl+Esc, go to All Programs -> Classic Shell -> Start Menu settings. Click All Settings radio button, then click 'Start Button' tab and make sure "Enable Start button" option is checked. Log off and log back in and see if the Start button returns.
If you see the Start button but it is taking you to Metro, from Settings, go to Controls tab and change "Left click opens" to Classic Start Menu.

Also, if it still doesn't show, have you any other Start menu replacement installed?

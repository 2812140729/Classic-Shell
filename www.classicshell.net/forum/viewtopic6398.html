Author:  Gaurav [ Mon Jan 14, 2013 11:57 pm ]
Post subject:  Re: Classic Shell stopped working
You can start Classic Start Menu from C:\Program Files\Classic Shell\ClassicStartMenu.exe. Doesn't starting that enable the Start button on your taskbar? If it does, then for some reason, ClassicStartMenu.exe is not automatically starting when you start Windows. Place it in your startup folder inside All programs.

Also, if you are familiar with Services.msc, check if the Classic Shell service is running and set it to Automatic.Sorry, I didn't realize you were running Windows 7. There is no service on Windows 7.

Post subject:  Re: Windows 10 - Blur Taskbar Using System Color
If you leave the taskbar color as default (right-click -> Set Default). Then the taskbar will try to match the system accent color. It will be correct, unless you turn off the colorization of the taskbar from the Windows settings, or the color is overridden by a registry key.

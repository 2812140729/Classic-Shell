Here is a plugin for Paint.net which allows opening and saving 32-bit BMP files (with alpha transparency). Extract the DLL to the Program Files\Paint.NET\FileTypes folder before starting Paint.net.

To avoid conflicts with the built-in Paint.net BMP functionality, the extension of 32-bit BMP files must be different. Only then will this plugin open them as 32-bit BMPs. For example, rename your file to Image.bmpx while opening and saving. Then from Explorer, you can rename it back to Image.bmp once you are done saving.

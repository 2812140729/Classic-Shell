Post subject:  Show full path and filename only in recent items hints?
When you hover over an item in the recent items menu you get a hint message in the format:

"Location: [FILENAME] ([PATH])"

If I'm hovering over an item it's usually because I'm trying to remember what/where it is, and the filename is already clearly visible, so it's really the path I'm after. And the word 'Location' isn't needed. A more useful format would be:

"[PATH]\[FILENAME].[EXTENSION]"

This would be expecially helpful when hovering over a bunch of files looking for one which is in a particular folder.

Post subject:  Re: Show full path and filename only in recent items hints?
The tooltip comes from Windows' Recent Items folder (C:\Users\<your user name>\AppData\Roaming\Microsoft\Windows\Recent) or shell:Recent. I don't know if any app except Explorer can modify it. Also, on my PC at least, it shows the full target path of the file but the folder path/location doesn't show the original file's location for me, it shows the location as Recent Items folder, so the folder path is useless.

Author:  Gaurav [ Thu Jan 23, 2014 10:22 pm ]
Post subject:  Re: How to get Windows Start Menu properties with a right-cl
Two ways - either Shift+right click on the Start button or open Classic Start Menu settings -> Controls tab. Check the option: "Right Click opens the Windows Context Menu" and click OK. If you check that then Shift+right click will open the Classic Shell context menu and regular right click will open the Windows Start button context menu.

I have added Icelandic (Iceland) translation to the three L10N ini files. The modified files are included in the ZIP file that is attached to this message.

To use Classic Shell in the Icelandic language:
  • Overwrite the L10N ini files in C:\Program Files\Classic Shell
  • Make sure you have set your operating system to use the Icelandic language.
  • Restart Classic Shell by signing out.

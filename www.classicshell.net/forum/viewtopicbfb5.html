Author:  Gaurav [ Fri Jan 18, 2013 1:24 am ]
Post subject:  Re: Option for including computer name on caption
%computername% is the environment variable you need to use. Just put it wherever you want. There are 3 places to put it - 'Menu caption', 'User name text' (both on Menu Look tab) and on the Start button tab, you can put %computername% in the 'Button tooltip'. If you want it in the caption including the OS name, just put the OS name followed by %computername%.

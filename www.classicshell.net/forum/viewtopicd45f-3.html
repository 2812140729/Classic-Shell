Post subject:  Re: I want 'recently openen' on taskbar but NOT in classic m
To turn off recently opened documents in the Start Menu, go to the General Behavior tab in Classic Start Menu settings (All Settings mode). Uncheck "Enable jumplists" option and click OK.

If you also want to remove the combined list of recent documents (called "Recent Items") from the Start Menu, go to the Special Items tab and set it to "Don't display this item".

All above settings will keep recently opened working on the taskbar.

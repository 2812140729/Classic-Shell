Post subject:  Experiment: CLASETRO [7style]
Inspired by Homer Simpson's snazzy car design, minus the tail fins.

I took the "Windows 8" skin, added very transparent classic 3d buttons, split dark mode,
part "Clear Sharp", a small set of Metro accent colors,
and some sun beams (reflection bands).

Ended up with...CLASETRO

It uses some Metro colors, so definitely won't work on Win7 operating system.

Really made it for Win10

Only works partially on Win8, because system lacks blur menu glass facility.
Could always install free trial Big Muscle for menu glass blur.
Also there isn't the matching Metro accent colors. I put in substitute Metro colors.

Thanks Saijin_Naib.
I used 2 background patterns from http://www.msfn.org/board/topic/173360- ... on-images/

Thanks also to Funky Monkey for Hash dots.

I can't remember where I got the Aero reflections from. Too many downloads.
The original WIN7 one was too blurry. They all got modified anyway to fit menu.


Edit: Added the other two views of menu look and added Custom Image option back in.
Edit: Minor update to fix pattern mask edge.

Time to get back to my electronic projects. Bye

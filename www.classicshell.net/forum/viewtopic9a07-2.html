Post subject:  Re: Skiping metro screen with Windows 8.1 and CS 4
First, looks like you are confusing the login screen with the metro screen. The setting in Classic Shell controls what happens AFTER you log in - either to go to the metro screen or to go to the desktop.

Second, the setting has been moved to the Windows 8.1 tab.

Third, the setting doesn't do much. It is tied to the Windows 8.1's own setting, which you can find in the Navigation tab in the taskbar properties.

Fourth, this thread describes how to disable the password screen: viewtopic.php?f=7&t=1318&view=unread#unread

Author:  Ivo [ Fri Oct 18, 2013 6:45 pm ]
Post subject:  Re: Skiping metro screen with Windows 8.1 and CS 4
The "blue background" is the start menu It looks like the Windows 8 skin with black buttons, black text, and disabled transparency.
If you don't like it you can change the skin settings or pick a different skin.

I am still not sure what you are asking about the metro screen.

If you mean the screen where you type your password, that is not the metro screen. That's the login screen. Classic Shell has no powers over that. If the instructions in my first reply don't work for you, then there is not much more I can suggest.

If you really mean the metro screen (the one with the colored tiles) and you want to skip it AFTER you log in, then look for the setting called "Skip Metro screen" in the "Windows 8.1 Settings" tab.

My suggestions for implementation
  1. Change the appearance of the avatar frame between the square and round stylized for Windows 10 and larger avatar frame
  2. Larger icons for items on the right side of the menu
  3. Icons on the left for the right-hand side menu names
  4. More predefined start buttons
  5. Ability to add your own shortcuts, location to the right of the menu
  6. You can add a command line to the right of the menu
  7. Possibility to change submenu to dark style
  8. Can set the same color and transparency instead of the white menu page
  9. Sort the Applications folder by name just like regular folders in the menu
  10. You can choose whether to use Modern UI applications in the Applications folder or as shortcuts

Post subject:  Re: Context menu -> Auto Arrange
Auto-arrange means - "from now on, always sort the items alphabetically". It is different from "Sort by Name" because that would only sort once and then append new items at the end.
It is a per-folder setting because you may want different folders to be arranged differently.

Hey Pipalou50.
It's a bit like Vista search box. I wanted to have blur transparent option,
but in order to have blur color on the first column, it makes blurry black text,
because Clear Type gets turned off when Fullglass opacity applied, so I skipped it.

After all these years we have been coming to this Classic Shell web site,
along with the experiments we made at skinning,
I'm still dumbfounded by the way glass color masks work.

There appear to be different ways of combining red (glass color),
and green(white color).

It's like there is a nonlinear, and linear curve way,
of adding glass/white proportions.

On this skin, I opted for adding small amounts of the green mask, to the high red mask value.
Example: 255 red with 104 green.
If I used 151 red with 104 green, the color saturation amount seen on screen would be different.

Here are different examples:
To get white color on screen, I can have 0 red mask with 255 green mask,
or 255 red mask with 255 green mask.
It looks like once total combined red green value goes past 255,
the parameters change to nonlinear curve, like a gamma curve.

Anyway, I don't want to play around with the skin anymore.
I probably should have kept the total combined red green mask values to be no more than 255,
for I see I keep getting the same bright color tint, even though the dark taskbar color tint changes.

Oh well, every skin is an experiment. Kind of like a lottery. Win a little, lose a lot.
As you can see by all my ramblings, I drank some strong coffee this morning.

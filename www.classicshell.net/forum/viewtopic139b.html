Due to a bug in Windows, sometimes File Explorer can get confused between addons, especially when there are multiple addons and one of the addons is updated or uninstalled & reinstalled. To fix this, open Registry Editor (Regedit.exe), go to HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Toolbar\ShellBrowser and delete the ITBar7Layout value. Thereafter, Explorer should not get confused which toolbar it enables/disables. After deleting the ITBar7Layout value, you must disable and enable the Classic Explorer toolbar once again in Explorer via the View menu (on Windows 7) or View tab of the Ribbon (Windows 8 and above).

The above steps must be done of course in addition to all the Classic Shell related addons being enabled via Internet Options -> Programs tab ->Manage Add-ons.

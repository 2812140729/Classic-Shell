Author:  Gaurav [ Wed Aug 03, 2016 1:09 pm ]
Post subject:  Re: classic start delete folder
That's because Windows tries to resolve the shortcut. Open these folders in Explorer:
C:\Users\<Your user name>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs
AND
C:\Users\<Your user name>\AppData\Roaming\ClassicShell\Pinned

Select the broken shortcut with left click and press Delete key on the keyboard.

(Appdata folder might be hidden on your PC. To see it, turn on hidden files and system files).

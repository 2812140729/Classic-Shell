Author:  Gaurav [ Wed Sep 21, 2016 7:04 am ]
Post subject:  Re: Impossible to remove windows explorer toolbar?
Yes you can turn off the Classic Explorer toolbar. If you are using Windows 8 or Windows 10, go to the View tab of the Ribbon and click on the dropdown arrow on the last button called Options. Uncheck "Classic Explorer Bar".

For Windows 7, the View menu -> Toolbars -> unchecking it can turn it off.

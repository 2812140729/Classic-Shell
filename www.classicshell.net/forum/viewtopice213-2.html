Author:  Gaurav [ Wed Sep 13, 2017 8:56 am ]
Post subject:  Re: Program is not working at all
Classic Shell does not calculate the reported value. It is all done by Explorer. Classic Shell merely shows it. For large files, Explorer takes time to calculate the size. If you check your 11.9 GB file again after a while, it should show the full size. You might be using a hard disk drive. On an SSD, the size calculation is almost instant.

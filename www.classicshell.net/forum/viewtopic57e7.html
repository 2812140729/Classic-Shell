On PCs which have IDT audio, the audio driver (IDTNC64.cpl file to be specific) is known to be incompatible with the latest Windows 10. It can cause Explorer/Classic Shell to crash. Uninstall IDT audio driver or get the updated fixed one. Simply removing IDTNC64.cpl should fix the crash too.

Author:  Ivo [ Fri Aug 26, 2016 1:47 pm ]
Post subject:  Re: ALL_PROGRAMS option, Skinning tutorial
If you are running on Windows 7, there will be an extra page in the settings to choose the skin options for the All Programs sub-menu. The "AllPrograms_options" determines what options will be visible in that page. Since only the sub-menu portion of the skin will be used, any options that only control the main menu should not be in the list.

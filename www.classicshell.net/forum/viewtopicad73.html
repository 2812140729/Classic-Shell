Hi. Not sure if OneSerendipity is around anymore. Been more than a year.
He made the original skin, at begining of the "Windows Vista Start Menu Skin" post.
He included a modified StartMenuL10N.ini file, that had extra letters for the shutdown button parts, to stretch it in some way.
Some of us got together, and improved the skin somewhat, where it did not need that language file to be modified anymore.

Opening up in Notepad, looking at the English part of that language file:

Menu.Shutdown = &Shut Downaaaliiiiiigi



should be like below:

Menu.Shutdown = Sh&ut Down

Can either edit the file, delete and replace the whole file with a default one,
or maybe easiest to uninstall Classic Shell, and reinstall Classic shell?
Logging out/restarting might be good to do too.

I have Classic Shell 4.0.4 (Classic Start, Classic Explorer, and Classic IE) installed on a Win 7 Pro (64 bit) computer I have. I am buying another Win 7 Pro (64 bit) computer, and plan to install Classic Shell on it.

I know that I can create Backup .xml files from Start, Explorer, and IE, and install them on the new computer. Are there any other files that I need to backup (or copy) to make sure that I have Classic Shell set up exactly the same on the new computer as the old one? Also, should I install the program before or after I install the backup files?

Author:  Ivo [ Thu Apr 17, 2014 9:56 am ]
Post subject:  Re: Buying a new computer
The XML files contain all the settings. There are other customizations (like what programs you have pinned, custom ordering of programs, etc) that are kept in the registry. However those don't generally translate from one computer to another because programs are often different or installed in different locations.

You need to install the software first before importing the XML files.

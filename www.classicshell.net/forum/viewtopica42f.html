Author:  Gaurav [ Sun Apr 30, 2017 12:06 pm ]
Post subject:  Re: Creators Update Stopped Classic Shell From Working
Open Classic Start Menu Settings using Windows Explorer (File Explorer) by going to C:\Program Files\Classic Shell\Start Menu Settings.

Check "Show all settings" at the top.

On the Controls tab, make sure that "Left Click opens" and "Windows Key opens" are both set to "Classic Start Menu".

Next go to General Behavior tab and make sure "Start automatically for this user" is enabled.

Author:  Ivo [ Sat May 06, 2017 7:43 pm ]
Post subject:  Re: Creators Update Stopped Classic Shell From Working
You have set ClassicStartMenu.exe to run as admin. It should not run as admin. Its main purpose is to hook into the Explorer process and should run with the same integrity level as Explorer.
Same for ClassicIE_64.exe

Also it appears you copied ClassicStartMenu.exe to the Startup folder instead of creating a shortcut there. The EXE can't run from that folder because it expects its other components to be in the same location.

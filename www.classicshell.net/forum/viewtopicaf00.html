Author:  lmh3 [ Thu May 11, 2017 6:54 am ]
Post subject:  Re: classic shell shrinks the taskbar
I have the same problem, Windows 7 x64, Classic Shell 4.3.0, classic start menu, no skin.
The taskbar shrinks after every reboot as depicted in first post.
The taskbar is on the left side of the screen, locked, autohide is on.
In Classic Shell settings "Customize taskbar" is off.
It may not directly relate to Classic Shell, but it seems to be triggered by it.

I can trigger the taskbar shrinking:
Go to Classic Shell settings / Taskbar
Click box "Customize Taskbar" on, no other changes, no need to click ok
Taskbar shrinks as depicted in first post

Also, with "Customize taskbar" on, unchecking the box (no need to click ok) will shrink the taskbar as well.

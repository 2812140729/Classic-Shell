Classic IE is just an addin for IE. To disable it, from Control Panel, open Internet Options -> Programs tab -> Manage Add-ons -> and disable the Classic Shell related addons. Then try if IE opens. If it still crashes, then the Classic Shell IE addin is not responsible for the crash.

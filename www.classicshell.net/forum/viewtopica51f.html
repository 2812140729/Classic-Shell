Post subject:  Windows 8.1 compatibility
Classic Shell 3.6.8 and later are compatible with Windows 8.1.


Additionally, Classic Shell 4.0.4 and later detect when you upgrade from Windows 8 to Windows 8.1 and offer to repair the installation automatically.

However, if for some reason, the Start Menu doesn't start automatically after you upgraded in-place to Windows 8.1 from Windows 8, you may need to do a Repair installation of the same version of Classic Shell that you have installed or upgrade the newest version. That should install all the missing bits again for Windows 8.1. This is necessary because during the upgrade, Windows setup resets some portion of the registry required for Classic Shell to start.

To do a Repair installation, download the installer of the Classic Shell version you have installed and choose "Repair".


If you are installing Classic Shell on a clean installation of Windows 8.1, then please download the latest version.


If the Start Menu does not start automatically at startup even after doing a Repair install, then just create a shortcut to C:\Program Files\Classic Shell\ClassicStartMenu.exe in your startup folder. The startup folder can be opened by typing 'shell:startup' into the Start Menu's search box and pressing Enter.

If your Windows 8.1 boots to the Metro UI, go to the Windows 8.1 Settings tab in Start menu Settings and check the option "Skip Metro screen", if it's not already checked.

So try to uninstall Classic Shell completely from Programs and Features Control Panel and reinstall. All your settings will be preserved.

That border is added by the skin itself, If you want to do it, you will need to open the .skin file with reshacker, export the appropriate bitmaps. (see: http://www.classicshell.net/tutorials/skintutorial.html for more info) modify, then re-import them. If you would like this done for you, attach the .skin file to this thread, or attach a link to download it. Then someone may decide to take on your request, though the modifications aren't too difficult in this instance.

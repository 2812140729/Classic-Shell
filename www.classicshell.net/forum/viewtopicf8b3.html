1 http://www.angusj.com/resourcehacker/ get this
2. browse to your skin folder in classic shells install dir
3. open the skin that matchs most closely to your desire *make note that .skin7 files are for the windows 7 style menu and .skin are for the other 2 styles* I would guess that your starting point will be Windows XP Luna.skin
4. The first file contains the code for the first skin variation (most skins only have 1, but some have 2-3 in this folder) locate "Main_bitmap=" in the (probably) the first file. it may appear more than once, depending on the settings you have. and take note of all the numbers next to this text
5. go to the bitmap folder, and export all the images matching these numbers
these images are designed to stretch in classic shell, as such the dimensions are important to maintain. so you should take this image, and re-color it to meet your needs
6. import this .bmp over all the necessary files

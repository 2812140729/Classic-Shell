One or two others were attempting the same thing with a 7 style Luna menu.
Does not look if it ever got completed?

Whenever modifying a menu, make sure to turn on "Report skin errors" in the "General Behavior" tab.
Sometimes CSM program will tell you if a wrong size bitmap is being used.
Most alignments have to be pixel perfect measurements, or could end up with a white skinless menu showing.
I find it's best to make a couple changes at a time, save it, and see what it looks like.
Save a copy in a safe place that won't get overwritten after a number a changes.
The worst thing to do, is make so many changes at once, without a backup copy.
Then one tries it and find it's all buggered up, without knowing what crippled skin.
I've done that many times, and paid the price for being a lazy backup bum.

As Jcee mentioned, going classic to 7 style is a lot more complex, with more bitmaps / text required.
Going other way CSM program just disregards 7 style bitmaps / text. There wont be any classic one column,
user picture out of whack, and some 7 style list text, has to be converted back to submenu text.


There are restrictions as to what 7 style can do.

Pluses and minuses.
Like no icons on second column

Has separate power button

No single column

Has a separate programs button
User picture is more or less stuck in center of second column.

User name can still work, just put in left column.

Instead of jump and search submenus, they become part of the main menu.
So three main menu bitmaps. Classic Luna (Ivo's), has a main bitmap for single, and another for two column.

The 7 style submenu is inside main menu, or can have fly out menu.

Due to a wide variety of computer sizes, with different dot per inch specs, you might get skin to work at standard 96 DPI,
then all alignment goes out on 120, 144, 192 DPI.
Ivo has made it so there is a 144 HDPI option that kicks in to use bigger images and alignments.

Can't have icons down at bottom like original XP menus.
There is a way to carefully put a picture of icon in the power button. It's very tricky to stop the image from stretching distortion.

Another way is have icon picture as an emblem, that can be placed close to shutdown button that shows on menu, or make shutdown arrow as icon picture.

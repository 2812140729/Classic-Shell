The idea behind this start button is that it lights up on a ten count indicating 'Go For Launch'. I was limited by what can be seen in a 40x40 pixel image. I tried a number of things and this was the result. Pretty close to what I intended. Counts 1 through 9 are in the outer ring. The tenth segment lights up the center and occurs when you press the button. I use a black windows theme and these colors go well with that.

This post inspired me to try and create something new:

http://www.classicshell.net/forum/viewtopic.php?f=18&t=4943

While obviously not the same, the author's creativity got me thinking about what I could do that I haven't yet thought of. This was the result.

Edit; Made some small tweaks to the geometry and color, picture has been updated (but has the same name as previously used).

Edit 2; Increased the resolution to 256x256, this allows you to scale the icon up and down better. I changed the colors so that the button will go with any windows color.

Edit 3; I deleted the original button. I prefer the alternate button much more. The alternate is available below.

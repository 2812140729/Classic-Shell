The appearance of other applications is not modified by any of these themes. But I am afraid, for Windows 8.1, I don't know any theme with:
1. Left-aligned window titles
2. Darker active window button and
3. Windows 7 appearance

There is only a Windows 7 theme for Windows 7 that I have with a darker active window and left aligned titles, as this screenshot shows.

Also note that themes/visual styles are very OS specific. A theme for Windows 8.1 won't correctly apply to Windows 8 and vice versa. Even the subsequent updates that Windows 8.1 got (Update 1, Update 2 and Update 3) caused themes to break so you'll need a theme specific to the last FIXABLE version of Windows, which is 8.1 Update 3 (November 2014 Update Rollup) before you can correctly apply these themes.

Splitwirez, your ClassicTheme.exe seems to work properly under Win81x64, except one thing: the taskbar becomes white, and the font color is white, too. I didn't find a way to change it. The same is under Win81x86.


Yeah, that happens because Windows gets a little confused as to how to apply the glass/accent transparency effect to the Classic Theme, so it just gets really bright instead, often appearing white. There are two ways to fix this;
1. Use the SetWindowCompositionAttribute tool from the Aeroglass 8.1+ website to remove both effects (this can be automated via AHK or a BAT file)
2. Use a black or extremely dark wallpaper...atleast, the part of the wallpaper behind the taskbar has to be black. The rest can be whatever you want.

... is a dumb way 'cause 3 sec may be too many or too little.
Code:
@echo off
echo/
if not exist "%~dp0SetWindowCompositionAttribute.exe" (
echo SetWindowCompositionAttribute.exe not found (must be in the same directory^).
exit /b -2
)
set Args=%*
if not defined Args (
echo Parameters for SetWindowCompositionAttribute.exe starting from the 3rd should be
echo specified.
echo/
echo Example:
echo %~nx0 accent 0 0 0 0
exit /b -1
)
>nul sc query ClassicTheme || (
echo ClassicTheme service is not running.
exit /b
)
color 00
:Loop
for /f "skip=1" %%A in ('tasklist /fi "imagename eq explorer.exe"') do (
"%~dp0SetWindowCompositionAttribute.exe" class Shell_TrayWnd %*
echo OK.
exit /b
)
if errorlevel 1 echo Waiting for explorer.exe to appear...
>nul ping -n 1 127.0.0.1
goto Loop

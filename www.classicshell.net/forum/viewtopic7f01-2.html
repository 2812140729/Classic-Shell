Post subject:  Uninstalling crashed my Win 10
I just love Classic Shell, but after the Windows 10 Creators update, it would always boot into the traditional windows 10 start-up menu and not Classic Shell. So if you then load Classic Shell settings, without doing anything, it suddenly would start working fine for that session. Until reboot.

So I thought to uninstall it and reinstall. Well, what a disaster that turned out to be!

Uninstall progressed fine until the point where it said "reboot your computer" - it then booted initially to log-on, then into a black screen with only a non-response start button visible. Flashed on and off in about 5 sec fashion, as if in a permanent loop, with my cooling fan running feverishly. System Task indicated 100% disk usage. Could not initiate a memory dump.

I then used advanced recovery to load the last system restore. Luckily Classic shell ran "Create a System restore point" prior to uninstall - thank you for that!

But, this incident has scared the life out of me and wasted 3 hours of my time. Hesitant to continue to use it again (or uninstall it?!)

I have Windows 10 64-bit system. Lenovo Z510. 12Gb or RAM.

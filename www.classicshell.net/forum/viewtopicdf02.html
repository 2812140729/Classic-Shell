Post subject:  Re: Open new window using CTRL click on parent button
On Windows 7, the Up button is added to Explorer by Classic Explorer. On Windows 8/8.1/10, Microsoft added it back themselves but forgot the Ctrl+click function.

So simply add the Up button to your Explorer toolbar and use it instead. Click+clicking it should open the parent folder in a new window. Ignore the Up button in the address bar.

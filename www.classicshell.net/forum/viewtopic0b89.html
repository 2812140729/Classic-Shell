Post subject:  Metro Color Tool (Win 7 style)
I made this Metro Color tool skin to help people understand Win 10,
which has special color tables, that it works from, to color the menu.
I don't know which ones are best. I included a bunch to pick as options.
There are lots more.
It would be too much to put every color as an option to try out.
If this wets your appetite, you can download the Classic Shell utility to see all the colors available.
You can make a personalized skin for yourself, and upload to show the world.


As this is a limited testing tool, you may find selector buttons too light, dark, or even disappear.
It's because I did not make color options for selector buttons. Separators stay same too. Fixed some of them.
I made it for better understanding of Ivos Metro skin, which this is modded from.

You can use it as a stand alone skin, if you want.
Upon info from Ivo, there can be 4 different colors on a bitmap mask. Black, Red, Green, and blue colors can be used as masks,
to reference any color available.
Can be a regular hex color example " #00FFFF " or a Metro color " $StartSelectionBackground ".
To make the Metro color work, the additional dollar sign " $ " is put in front of Metro color text.

I made skin to have a gradient bar at top, and left and right columns with different colors.
You can pick different colors for left top bar, and right top bar to give it a gradient. Or just make every color same.
Up to you to experiment.


EDIT: When Windows introduced Dark theme in update, it changed what was black to white.


To see black as black, turn off Dark app mode in Windows color settings.

There are quite a few issues related to the current version of windows 10, and that will likely compound further; because classic shell doesn't seem to be in active development anymore. Personally it works flawless (on the latest version of windows 10) but there have been various issues cropping up. (more than normal) due in-part to the hectic development cycle Microsoft is transitioning to for windows 10. (coupled with how deeply classic shell is rooted in the system)
If you stop at the (as of now) latest version; chances are classic shell will work for you just fine. However its highly likely something might break classic shell for you in the future, and it may never get fixed.

Windows 8.1 is your best bet if you want longevity in classic shell coupled with all the necessary updates.

Author:  Gaurav [ Fri Jan 18, 2013 7:27 pm ]
Post subject:  Re: Programs Menu
In XP, it does not open in the same column. It opens to the right in a different column. The Vista/7 style is inefficient because it restricts All Programs to a smaller area which the user has to scroll and requires clicking each subfolder to launch a program. Compared to that, the flyout style has better usability, no scrolling required as it shows in multiple columns, and no clicking required, only hover is enough. So right now, there are no plans to add the Vista/7 style to Classic Shell.

To have the name of the Windows version on the left, go to Skin tab and check the 'Caption' option. It is only available in the single column style, as the Windows menu did not have caption in dual column style.

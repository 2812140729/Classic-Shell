hit win+R to open the run dialog
type cmd
then in the command prompt window type
sfc /scannow /f /r
then press enter
let it do its thing, and it will hopefully fix your problems

The next step is to try system restore/reset (if the previous didn't solve your issues)

Hit win+I
Go to Update and security at the bottom
then recovery
There are a few options there that might help
reset this PC is the most straightforward and likely to fix the issue, however it will remove all installed programs (it will keep any of your documents/ pics though if you tell it to)

Then there is advanced start up options which will allow you to run system restore if you have any restore points. This will 'undo' recent changes since the last restore point. you may loose a few programs if they were installed after the restore point, but you wont loose most of them.

I was able to open Control Panel, then All Control Panel Items, then Default Programs on the left side (the navigation pane), then click on Select Folder.
The link it gave me is ::{26EE0668-A00A-44D7-9371-BEB064C98683}\0\::{17CD9488-1228-4B2F-88CE-4298E93E0966}
You can directly paste it.
The benefit of doing it this way over "control /name ..." is that you automatically get the correct name and icon.

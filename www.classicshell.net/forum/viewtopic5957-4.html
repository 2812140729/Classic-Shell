Which OS are you on? On Windows 7, you can use the Classic Windows Theme by right clicking the Desktop -> Personalize. The classic theme will give you 1. Matching background taskbar color as the classic Start button 2. Black text in notification area.

On Windows 7, you can customize colors of the classic theme by running:
rundll32.exe shell32.dll,Control_RunDLL desk.cpl,,5

On Vista, this command works: rundll32.exe shell32.dll,Control_RunDLL desk.cpl,,2
And then click the Advanced button.


Also, besides setting the Classic Shell Start Menu style to Classic, you can switch to the Classic skin from the Skins tab. Check "Solid Color Selection" if you want.

On Windows 8/8.1, there is no way to change colors of window elements because of the removal of the classic theme. All Windows 8 themes are based on visual styles. Unless you patch Windows to enable use of custom visual styles and then use a visual style made to look like Classic, and still the results won't be satisfactory.

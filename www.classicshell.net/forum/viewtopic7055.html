The background is set by lines like this:
Main_bitmap=$StartBackground

That means the main menu background will use the start background color. The taskbar uses the $SystemAccentDark2 or $SystemAccentDark3 color depending if it is transparent or not. You can try changing the menu to
Main_bitmap=$SystemAccentDark2

However, the selection also uses SystemAccentDark2, so you'll have to change that color as well. For example
Main_selection=$StartSelectionBackground

BTW, those are just examples. You will have to change multiple lines in the skin because the same colors are used in multiple places.

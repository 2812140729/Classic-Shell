Post subject:  Program list not automatically loading after Creator Update
I have updated several machines with the Windows 10 Creator Update. When the Classic Shell menu button is clicked, the menu appears but the area showing the list of programs on the left hand side of the menu is blank. Hovering over the All Programs button for 1 to 2 seconds makes the program list appear. Likewise clicking on the All Programs button also works. I have uninstalled Classic Shell, restarted the PC and re-installed the program. The problem still exists. Other than this issue, the Classic Shell functions I use seem to work correctly. It affects all the machines I have updated.

Have any other Classic Shell users seen this problem? I can live with this issue which is a nuisance but a fix would be nice.
Author:  Gaurav [ Sun Apr 09, 2017 10:54 pm ]
Post subject:  Re: Program list not automatically loading after Creator Upd
For Windows 10 Creators Update, there is a new setting under Settings -> Privacy: "Let Windows track app launches to improve start and search results". Turn it on.

Then go to Settings -> Personalization -> Start and turn on "Show most used apps". After turning it on, try launching a couple of programs and they should show up in the Frequent list.

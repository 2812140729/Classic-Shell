LCARS Tactical [7style] for CSM version 4.3.0



Menu skin was made from elements of Starship Atlantis wallpaper, by TronTrek:

http://fav.me/d6kbo1l

It is different from most LCARS wallpapers, due to the extra blue grey colors,
which makes it easier on the eyes, but harder to find matching wallpapers.

Variation 1 is your typical menu that can be adjusted.


Variation 2, 3, 4 use picture emblems for the colored bars on 2nd column.
Due to this fact, I have to stress these skin variations will be unusable if any of below requirements do not match:

HAS TO BE 100%, or 150% size. 125% is NOT available

HAS TO BE default settings with 13 menu item entries in right column. Was made for Windows 10, but can work for Win7/8.
Just need to go to "Customize Start Menu" tab to set up qty 13 entries to be displayed. 13 is the lucky number.
It will go out of alignment, and look ugly if other qty.

Sorry, one more, and I want to kick myself, for I thought it was default in all operating systems.
This font, Franklin Gothic Medium Cond, HAS TO BE installed. It squeezes up the text a little bit.
Without this font, bars go out of alignment, as the text padding is set for it exactly.

I initially tried Lcars, Steelfish Rg fonts, but were way too ffuzzzyy to use at such small font sizes. Too bad.

The Segoe font option does not work when the multicolor selector bars in 2nd column are showing.

Have only tested skin in English. I don't know if other languages change alignment.


Note. If this .skin7 is renamed to .skin it will show up in Classic 2 style, and can work.
On two of the variations there can be small icons on 2nd column.
The "Regular" variation can have large icons.

Additional note. I may have buggered up the 2nd column icons text when using Classic 2 style.
You might see white folders instead of the regular icons.
Click the small icons, and no icons check boxes a couple times to refresh back to normal.

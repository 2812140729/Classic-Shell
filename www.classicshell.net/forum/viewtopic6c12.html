The rules of what to show in the frequently used list are quite complex. In a nutshell, only programs with shortcuts in the start menu are shown, and only after you use that shortcut from the start menu at least once. After that the frequency of use is tracked regardless if you start the program from the start menu, the desktop, quick launch, or whatever. Programs pinned to the taskbar or the start menu are ignored.
That's how the system start menu behaves in Windows 7.

@waltz, which version of Classic Shell are you using? By root level you probably mean the Start Menu folder. If that folder is shown as inside All Programs (which is the case when the 'Pinned Programs folder' setting is set to "Use Pinned folder", then the Start Menu folder items *are* shown in the Recent list. In the other scenario, the Start Menu folder's contents are directly shown in the main menu. Why then would you want the same program again in the Recent list?

As for other folders, if you want to see anything in the Recent list, then add it as a custom item and check the option "Track recent items" for it.

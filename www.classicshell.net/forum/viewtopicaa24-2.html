Hello,

After running with the new Windows Start menu for the last few months, which although good in many ways, is severely hobbled by the lack of a fully-hierarchical All Programs menu.

So I've come back to Classic Shell and have managed to tweak a whole load of stuff to get me something very, very close (for Classic Shell) to the native look and feel.

One thing that doesn't really work though, is the "Show icon frames" option. All it seems to do is put a slight border around the icon, and slightly darken the background. This may be useful in other themes, but for a Windows 10 look, it doesn't work. Seeing as Microsoft have now finally really nailed the flat look in Windows 10, I suggest that this could be improved to more accurately reflect the native start menu.

I would happily not use it, but the extra padding it affords icons does it make it a little easier to view.

I'd also suggest (though not knowing anything about theme editing) that further settings (such as padding, icon background color) should be added to the Windows 10 Setting tab to allow a more faithful rendering of the new Start Menu (which actually looks pretty great).

Just my 2c,

Thanks,
Dave

The frames should not disappear if you turn off the transparency. The two settings are unrelated. You may have also unchecked the icon frames setting.

The current frame color is selected to work for all possible color schemes. If you want it to be gray, you will have to edit the skin file. It is also possible to increase the size of the background, of course you will be able to fit fewer items on screen.

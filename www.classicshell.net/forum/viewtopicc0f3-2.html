Author:  juniper7 [ Fri Jul 07, 2017 10:17 pm ]
Post subject:  Re: Square version of Windows 7 Style Start Menu Request  
Here.
Only made square corners. I did not flatten out the selector buttons.
I never tested on other systems, besides Win 10.
The skinned scroll bar is not going to work with Creators Update (Win 10).

I didn't put in a switchable opaque/transparency option,
because just having option text,:

OPTION NOT_TRANS="Opaque",0

[NOT_TRANS]
Main_opacity=region
Main2_opacity=region

will make menu a lot darker, even black. It would take a total redesign of bitmaps, and color masks to look brighter.
Depending on which operating system, you have CSM set up on,
it might show a setting "Glass opacity". Set it to 100 for a bright opaque look,
or 25 for more transparency.

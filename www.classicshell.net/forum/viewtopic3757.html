Here’s what’s new:

  • Improved support for Windows 10
  • New ability to save settings to XML files from command line: http://www.classicshell.net/faq/#install_xml
  • New skin features, including glass for Windows 10, shadows for the text and fractional font sizes like 9.5
  • New setting on the 'Menu Look' tab to control the opacity of the glass on Windows 10
  • New setting to align the custom start button to the corner instead of the middle of the taskbar
  • Multiple minor improvements and bugfixes

You can download it from here: http://www.mediafire.com/download/06l6e ... _4_2_2.exe

Author:  Gaurav [ Wed Mar 16, 2016 2:37 am ]
Post subject:  Re: Classic Shell 4.2.2 beta is released
@Valle, All apps from the shell:AppsFolder namespace extension are shown. If the Calculator doesn't show up there, it is not installed in your Windows. As for the icons, you can right click the Start button and choose Exit. Then go to %localappdata%\ClassicShell and delete DataCache.db. Launch the Start Menu again from C:\Program Files\Classic Shell\ClassicStartMenu.exe. See if the icons show correctly now.

I stand corrected.
If on Windows 10, one can give a subtle faint color to menu and taskbar,
with the CSM Override glass color settings.

To get a pale powder purple:
*set skin option to transparent.
*In Menu look tab, set override to FF0000
*set Glass Opacity to 60

*Link the included taskbar image.
*Set to Glass Taskbar look.
*Set Taskbar Opacity to 75 and color same FF0000
*Border size set 2,0 stretch.

Max color intensity, which is not much, as there is no color mask in skin,
is made with Glass opacity set to 100, and Taskbar opacity set to 50.
It almost looks like some of the default color settings in Windows 7.


Blue FF8000 is gentle as well.

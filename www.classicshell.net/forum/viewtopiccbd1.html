Post subject:  Mod of Windows Aero.skin7 with red Shutdown button
Just a simple mod of the Windows Aero skin in Windows 7 style with the following modifications:


- Shutdown button background is red and shutdown submenu arrow background is blue, green when hovered and yellow when expanded. It also has a Power icon
- Green arrow for All Programs when inline treeview style of All Programs is used
- If you enable cascading All Programs submenu, then you will get a nice looking icon
- Font is Segoe UI Semibold because it is more readable in my opinion

Thanks to Splitwirez for the idea and juniper7 for the graphics

Install this skin like any other skin:
1. Download the file to your Desktop, Downloads or Documents folder.
2. Open File Explorer and copy the Windows Aero Mod.skin7 file from where you downloaded it to C:\Program Files\Classic Shell\Skins
3. Open Start Menu Settings and switch to the Skin tab.
4. Select the skin from the dropdown list and click OK.

Because this is a .skin7 format, it will only work for Windows 7 style of the menu.
